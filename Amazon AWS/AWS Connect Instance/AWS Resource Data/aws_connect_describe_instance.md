# Introduction <a name="introduction"></a>
The API is used to get the details of an Amazon Connect instance.
The response includes information such as instance status, creation time, service role, identity management settings, and overall configuration of the Connect instance.

The API belongs to the device type <b>AWS Connect</b> and uses the <b>connect</b> resource type of the AWS REST API.
