# Introduction <a name="introduction"></a>
The API is used to get the details of an Amazon Neptune DB cluster.
The response includes information such as cluster status, endpoints, engine version, replication configuration, storage settings, backup retention, and member instances.

The API belongs to the device type <b>AWS Neptune</b> and uses the <b>neptune</b> resource type of the AWS REST API.
