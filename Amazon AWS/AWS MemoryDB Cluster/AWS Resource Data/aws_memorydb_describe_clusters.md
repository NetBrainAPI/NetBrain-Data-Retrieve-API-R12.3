# Introduction <a name="introduction"></a>
The API is used to get the details of Amazon MemoryDB clusters.
The response includes information such as cluster status, node type, number of shards, parameter groups, snapshot settings, engine version, and Multi-AZ configuration.

The API belongs to the device type <b>AWS MemoryDB</b> and uses the <b>memorydb</b> resource type of the AWS REST API.
