# Introduction <a name="introduction"></a>
The API is used to retrieve all DNS record sets within a specified Amazon Route 53 hosted zone.
The response includes information such as record types, names, routing policies, target values, alias configurations, and TTL (Time-To-Live) settings.

The API belongs to the device type <b>AWS Route 53</b> and uses the <b>route53</b> resource type of the AWS REST API.
