# Introduction <a name="introduction"></a>
The API is used to get the details of all AWS Batch job definitions.
The response includes information such as container properties, retry strategies, resource requirements, job role settings, and versioned job definition metadata.

The API belongs to the device type <b>AWS Batch</b> and uses the <b>batch</b> resource type of the AWS REST API.
