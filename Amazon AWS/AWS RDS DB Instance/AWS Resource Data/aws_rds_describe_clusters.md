# Introduction <a name="introduction"></a>
The API is used to get detailed information about Amazon Aurora and Amazon RDS DB clusters.
The response includes configuration details such as cluster status, engine type and version, endpoints, storage settings, backup configurations, member instances, and Multi-AZ deployment attributes.

The API belongs to the device type <b>AWS RDS</b> and uses the <b>rds</b> resource type of the AWS REST API.
