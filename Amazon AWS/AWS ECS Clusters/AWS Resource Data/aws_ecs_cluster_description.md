# Introduction <a name="introduction"></a>
The API is used to get the details of Amazon Elastic Container Service (ECS) clusters.
The response includes information such as cluster status, registered container instances, running tasks, capacity providers, settings, and overall cluster configuration.

The API belongs to the device type <b>AWS ECS</b> and uses the <b>ecs</b> resource type of the AWS REST API.
