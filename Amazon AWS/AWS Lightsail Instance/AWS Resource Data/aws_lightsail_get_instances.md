# Introduction <a name="introduction"></a>
The API is used to get the details of Amazon Lightsail instances.
The response includes information such as instance state, blueprint and bundle details, public and private networking information, attached disks, metrics configuration, and compute resource specifications.

The API belongs to the device type <b>AWS Lightsail</b> and uses the <b>lightsail</b> resource type of the AWS REST API.
