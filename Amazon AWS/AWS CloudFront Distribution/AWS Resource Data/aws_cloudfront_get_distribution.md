# Introduction <a name="introduction"></a>
The API is used to get the details of an Amazon CloudFront distribution.
The response includes information such as distribution configuration, enabled status, origins, cache behaviors, geographic restrictions, SSL/TLS settings, logging configuration, and deployment state.

The API belongs to the device type <b>AWS CloudFront</b> and uses the <b>cloudfront</b> resource type of the AWS REST API.
