# Introduction <a name="introduction"></a>
The API is used to get the details of Amazon ElastiCache cache clusters.
The response includes information such as cluster engine type, node configurations, parameter groups, snapshot settings, network details, maintenance windows, and overall cache cluster status.

The API belongs to the device type <b>AWS ElastiCache</b> and uses the <b>elasticache</b> resource type of the AWS REST API.
