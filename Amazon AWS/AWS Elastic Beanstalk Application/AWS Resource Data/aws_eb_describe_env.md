# Introduction <a name="introduction"></a>
The API is used to get the health status of an AWS Elastic Beanstalk environment.
The response includes details such as overall health state, color indicators, causes of degraded performance, instance health summaries, environment resources, and operational metrics that describe the current state of the environment.

The API belongs to the device type <b>AWS Elastic Beanstalk</b> and uses the <b>elasticbeanstalk</b> resource type of the AWS REST API.
