# Introduction <a name="introduction"></a>
The API is used to get the details of AWS Elastic Beanstalk applications.
The response includes information such as application names, creation timestamps, description metadata, resource lifecycle settings, and configuration templates associated with each application.

The API belongs to the device type <b>AWS Elastic Beanstalk</b> and uses the <b>elasticbeanstalk</b> resource type of the AWS REST API.
