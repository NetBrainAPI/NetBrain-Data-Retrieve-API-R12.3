# Introduction <a name="introduction"></a>
The API is used to get detailed information about an AWS Storage Gateway appliance.
The response includes gateway type, operational state, connectivity status, timezone settings, last software update, and general configuration details of the gateway.

The API belongs to the device type <b>AWS Storage Gateway</b> and uses the <b>storagegateway</b> resource type of the AWS REST API.
