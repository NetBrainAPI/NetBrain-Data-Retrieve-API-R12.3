# Introduction <a name="introduction"></a>
The API is used to retrieve details of Amazon DocumentDB (DocDB) clusters.
The response includes configuration information such as cluster status, engine version, endpoint details, storage configuration, member instances, and cluster parameter settings.

The API belongs to the device type <b>AWS DocumentDB</b> and uses the <b>docdb</b> resource type of the AWS REST API.
