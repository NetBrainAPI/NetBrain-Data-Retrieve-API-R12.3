# Introduction <a name="introduction"></a>
The API is used to list all services associated with a specific AWS VPC Lattice Service Network.
The response includes information such as service identifiers, association status, creation timestamps, and metadata describing how services are linked to the service network.

The API belongs to the device type <b>AWS VPC Lattice</b> and uses the <b>vpc-lattice</b> resource type of the AWS REST API.
