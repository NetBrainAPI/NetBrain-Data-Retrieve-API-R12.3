# Introduction <a name="introduction"></a>
The API is used to list all VPC endpoint associations linked to an AWS VPC Lattice Service Network.
The response includes information such as VPC endpoint identifiers, association status, creation timestamps, and metadata describing how VPC endpoints are connected to the service network for traffic routing.

The API belongs to the device type <b>AWS VPC Lattice</b> and uses the <b>vpc-lattice</b> resource type of the AWS REST API.
