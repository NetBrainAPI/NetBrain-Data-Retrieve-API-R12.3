# Introduction <a name="introduction"></a>
The API is used to list the listeners configured for an AWS VPC Lattice service.
The response includes details such as listener protocols, ports, associated rules, rule priorities, and overall listener configuration for the service.

The API belongs to the device type <b>AWS VPC Lattice</b> and uses the <b>vpc-lattice</b> resource type of the AWS REST API.
