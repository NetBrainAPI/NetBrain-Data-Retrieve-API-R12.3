# Introduction <a name="introduction"></a>
The API is used to get detailed configuration information for an AWS Lambda function.
The response includes details such as runtime, handler, memory size, timeout settings, execution role, environment variables, VPC configuration, layers, dead-letter configuration, and versioning details.

The API belongs to the device type <b>AWS Lambda</b> and uses the <b>lambda</b> resource type of the AWS REST API.
