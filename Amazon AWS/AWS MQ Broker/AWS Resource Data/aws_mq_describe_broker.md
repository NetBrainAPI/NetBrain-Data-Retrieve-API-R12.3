# Introduction <a name="introduction"></a>
The API is used to get the details of an Amazon MQ Broker.
The response includes information such as broker status, deployment mode, engine type and version, host instances, security settings, storage configurations, and network endpoints.

The API belongs to the device type <b>AWS MQ</b> and uses the <b>mq</b> resource type of the AWS REST API.
