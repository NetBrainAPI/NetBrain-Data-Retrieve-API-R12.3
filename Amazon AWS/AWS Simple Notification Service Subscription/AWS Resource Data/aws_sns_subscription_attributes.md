# Introduction <a name="introduction"></a>
The API is used to get the details of Amazon SNS subscription attributes.
The response includes information such as subscription protocol, endpoint, delivery policies, filter policies, confirmation status, raw message delivery settings, and ownership details.

The API belongs to the device type <b>AWS SNS</b> and uses the <b>sns</b> resource type of the AWS REST API.
