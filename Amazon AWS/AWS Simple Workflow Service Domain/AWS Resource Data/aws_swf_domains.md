# Introduction <a name="introduction"></a>
The API is used to get the details of an Amazon Simple Workflow Service (SWF) domain.
The response includes information such as domain status, retention period settings, workflow execution configurations, description metadata, and domain-level policies.

The API belongs to the device type <b>AWS SWF</b> and uses the <b>swf</b> resource type of the AWS REST API.
