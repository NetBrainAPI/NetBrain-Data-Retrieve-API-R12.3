# Introduction <a name="introduction"></a>
The API is used to get the details of Amazon FSx file systems.
The response includes information such as file system type, storage capacity, lifecycle state, network configuration, deployment type, throughput capacity, and administrative settings.

The API belongs to the device type <b>AWS FSx</b> and uses the <b>fsx</b> resource type of the AWS REST API.
