# Introduction <a name="introduction"></a>
The API is used to get the details of an AWS Global Accelerator accelerator.
The response includes information such as accelerator status, DNS name, IP address type, enabled state, associated listeners, and overall configuration settings.

The API belongs to the device type <b>AWS Global Accelerator</b> and uses the <b>globalaccelerator</b> resource type of the AWS REST API.
