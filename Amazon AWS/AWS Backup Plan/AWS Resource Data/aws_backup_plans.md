# Introduction <a name="introduction"></a>
The API is used to get the details of an individual AWS Backup plan.
The response includes information such as backup rules, target resources, lifecycle configurations, and advanced backup settings defined in the plan.

The API belongs to the device type <b>AWS Backup</b> and uses the <b>backup</b> resource type of the AWS REST API.
