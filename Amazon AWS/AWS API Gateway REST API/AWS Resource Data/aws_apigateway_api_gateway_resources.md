# Introduction <a name="introduction"></a>
The API is used to retrieve and check the overall resources of an Amazon API Gateway REST API.
The response includes details of the resource hierarchy, methods, and associated configurations that define the REST API structure.

The API belongs to the device type <b>AWS API Gateway</b> and uses the <b>apigateway</b> resource type of the AWS REST API.
