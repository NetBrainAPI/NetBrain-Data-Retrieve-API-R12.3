# Introduction <a name="introduction"></a>
The API is used to get all the express route circuits in the current subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Express Route Circuits - List All>. 



For a complete list of available API resources for Azure Express Route Circuits, please refer to Microsoft document: [Express Route Circuits - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Express Route Circuits - List All - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits/list-all?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2024-05-01&tabs=HTTP)
