# Introduction <a name="introduction"></a>
The API is used to get all the stats from an express route circuit in a resource group from Azure API Server (https://management.azure.com/). The response includes the details like primarybytesIn, primarybytesOut, secondarybytesIn and secondarybytesOut. 



The original Azure RESTful API used is <Express Route Circuits - Get Stats>. 



For a complete list of available API resources for Azure Express Route Circuits, please refer to Microsoft document: [Express Route Circuits - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Express Route Circuits - Get Stats - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits/get-stats?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2024-05-01&tabs=HTTP)
