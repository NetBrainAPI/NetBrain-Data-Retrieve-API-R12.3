# Introduction <a name="introduction"></a>
The API is used to get the Azure MSEE expressroute circuit information from Azure API Server (https://management.azure.com/). The response includes the details like primaryBytesIn, primaryBytesOut, secondaryBytesIn and secondaryBytesOut.

The original Azure RESTful API used is <Express Route Circuits - Get>. 



For a complete list of available API resources for Azure Express Route Circuits, please refer to Microsoft document: https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2023-09-01

For the original Azure API detailed information, please refer to Microsoft document: https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits/get?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2023-09-01&tabs=HTTP
