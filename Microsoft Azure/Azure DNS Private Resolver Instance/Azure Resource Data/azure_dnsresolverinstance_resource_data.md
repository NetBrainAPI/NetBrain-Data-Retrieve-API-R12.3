# Introduction <a name="introduction"></a>
The API is used to get the Azure DNS Private Resolver Instance resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Dns Resolvers - Get>. 



For a complete list of available API resources for Azure DNS Private Resolver Instances, please refer to Microsoft document: [Dns Resolvers - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolvers?view=rest-dns-dnsresolver-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Dns Resolvers - Get - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolvers/get?view=rest-dns-dnsresolver-2025-05-01&tabs=HTTP)
