# Introduction <a name="introduction"></a>
The API is used to retrieve a list of all IP prefixes that azure firewall has learned to not SNAT from Azure API Server (https://management.azure.com/). The response includes the details like ipPrefixes.



The original Azure RESTful API used is <Azure Firewalls - List Learned Prefixes>. 



For a complete list of available API resources for Azure Firewall , please refer to Microsoft document: [Azure Firewalls - REST API (Azure Firewall) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/firewall/azure-firewalls?view=rest-firewall-2025-03-01&viewFallbackFrom=rest-firewall-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Azure Firewalls - List Learned Prefixes - REST API (Azure Firewall) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/firewall/azure-firewalls/list-learned-prefixes?view=rest-firewall-2025-03-01&viewFallbackFrom=rest-firewall-2024-05-01&tabs=HTTP)
