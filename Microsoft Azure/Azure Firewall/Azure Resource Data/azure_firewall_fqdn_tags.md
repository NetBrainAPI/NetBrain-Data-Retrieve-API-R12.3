# Introduction <a name="introduction"></a>
The API is used to get all the Azure Firewall FQDN Tags in a subscription from Azure API Server (https://management.azure.com/). The response includes the details of vpn connection such as name, id, type, location, tags, properties, etc.
<br>
<br>


The original Azure RESTful API used is <Azure Firewall Fqdn Tags - List All>. 
<br><br>


For a complete list of available API resources for Azure Firewall, please refer to Microsoft document: [Azure Firewall Fqdn Tags - REST API (Azure Firewall) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/firewall/azure-firewall-fqdn-tags?view=rest-firewall-2024-07-01&viewFallbackFrom=rest-firewall-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Azure Firewall Fqdn Tags - List All - REST API (Azure Firewall) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/firewall/azure-firewall-fqdn-tags/list-all?view=rest-firewall-2024-07-01&viewFallbackFrom=rest-firewall-2024-05-01&tabs=HTTP)
