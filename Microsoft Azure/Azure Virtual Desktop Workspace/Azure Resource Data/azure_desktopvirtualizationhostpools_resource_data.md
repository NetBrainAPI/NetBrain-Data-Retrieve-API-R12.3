# Introduction <a name="introduction"></a>
The API is used to get the Azure Virtual Desktop Workspace resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Workspaces - Get>. 



For a complete list of available API resources for Azure Virtual Desktop Workspace, please refer to Microsoft document: [Workspaces - REST API (Azure Desktop Virtualization) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/desktopvirtualization/workspaces?view=rest-desktopvirtualization-2024-04-03)

For the original Azure API detailed information, please refer to Microsoft document: [Workspaces - Get - REST API (Azure Desktop Virtualization) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/desktopvirtualization/workspaces/get?view=rest-desktopvirtualization-2024-04-03&tabs=HTTP)
