# Introduction <a name="introduction"></a>
The API is used to get the Azure Private DNS Zone resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Private Zones - Get>. 



For a complete list of available API resources for Azure Private DNS Zones, please refer to Microsoft document: [Private Zones - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/privatedns/private-zones?view=rest-dns-privatedns-2018-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Private Zones - Get - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/privatedns/private-zones/get?view=rest-dns-privatedns-2018-09-01&tabs=HTTP#code-try-0)
