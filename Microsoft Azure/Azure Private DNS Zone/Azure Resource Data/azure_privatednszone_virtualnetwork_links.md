# Introduction <a name="introduction"></a>
The API is used to list the virtual network links to the specified Private DNS zone from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Virtual Network Links - List>. 



For a complete list of available API resources for Azure Private DNS Zones, please refer to Microsoft document: [Virtual Network Links - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/privatedns/virtual-network-links?view=rest-dns-privatedns-2018-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Network Links - List - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/privatedns/virtual-network-links/list?view=rest-dns-privatedns-2018-09-01&tabs=HTTP#code-try-0)
