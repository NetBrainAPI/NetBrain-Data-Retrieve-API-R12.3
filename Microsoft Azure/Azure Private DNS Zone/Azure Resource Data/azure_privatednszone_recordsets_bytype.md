# Introduction <a name="introduction"></a>
The API is used to list the record sets of a specified type in a Private DNS zone from Azure API Server (https://management.azure.com/). The response includes the details like id, type, name, properties, etc.



The original Azure RESTful API used is <Record Sets - List By Type>. 



For a complete list of available API resources for Azure Private DNS Zones, please refer to Microsoft document: [Record Sets - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/privatedns/record-sets?view=rest-dns-privatedns-2018-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Record Sets - List By Type - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/privatedns/record-sets/list-by-type?view=rest-dns-privatedns-2018-09-01&tabs=HTTP#code-try-0)
