# Introduction <a name="introduction"></a>
The API is used to list all available dedicated host sizes to which the specified dedicated host can be resized from Azure API Server (https://management.azure.com/). The response includes the details like dedicated host sizes.



The original Azure RESTful API used is <Dedicated Hosts - List Available Sizes>. 



For a complete list of available API resources for Azure Dedicated Hosts, please refer to Microsoft document: [Dedicated Hosts - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/dedicated-hosts?view=rest-compute-2024-11-04)

For the original Azure API detailed information, please refer to Microsoft document: [Dedicated Hosts - List Available Sizes - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/dedicated-hosts/list-available-sizes?view=rest-compute-2024-11-04&tabs=HTTP#code-try-0)
