# Introduction <a name="introduction"></a>
The API is used to get the Azure ExpressRoute Circuit resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Express Route Circuits - Get>. 



For a complete list of available API resources for Azure ExpressRoute Circuit, please refer to Microsoft document: [Express Route Circuits - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2025-01-01)

For the original Azure API detailed information, please refer to Microsoft document: [Express Route Circuits - Get - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-circuits/get?view=rest-expressroute-2025-03-01&viewFallbackFrom=rest-expressroute-2025-01-01&tabs=HTTP#code-try-0)
