# Introduction <a name="introduction"></a>
The API is used to get the Azure App Configuration resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Virtual Hubs - Get>. 



For a complete list of available API resources for Azure Route Server, please refer to Microsoft document: [Virtual Hubs - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hubs?view=rest-virtualwan-2025-03-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Hubs - Get - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hubs/get?view=rest-virtualwan-2025-03-01&tabs=HTTP#code-try-0)
