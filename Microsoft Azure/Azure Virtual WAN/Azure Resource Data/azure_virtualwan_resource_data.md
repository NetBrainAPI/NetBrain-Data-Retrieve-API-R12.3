# Introduction <a name="introduction"></a>
The API is used to get the Azure Virtual Machine Scale Set resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Virtual Machine Scale Sets - Get>. 



For a complete list of available API resources for Azure Virtual Machine Scale Set, please refer to Microsoft document: [Virtual Wans - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-wans?view=rest-virtualwan-2025-03-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Wans - Get - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-wans/get?view=rest-virtualwan-2025-03-01&tabs=HTTP)
