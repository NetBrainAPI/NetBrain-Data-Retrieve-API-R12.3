# Introduction <a name="introduction"></a>
The API is used to get all nat gateways in the current resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Nat Gateways - List>. 



For a complete list of available API resources for Azure NAT Gateway , please refer to Microsoft document: [Nat Gateways - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/nat-gateways?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Nat Gateways - List - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/nat-gateways/list?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-05-01&tabs=HTTP)
