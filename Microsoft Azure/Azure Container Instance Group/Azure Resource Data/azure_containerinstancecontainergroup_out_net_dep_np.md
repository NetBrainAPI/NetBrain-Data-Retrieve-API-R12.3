# Introduction <a name="introduction"></a>
The API is used to get all the network dependencies for this container group to allow complete control of network setting and configuration information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Container Groups - Get Outbound Network Dependencies Endpoints>. 



For a complete list of available API resources for Azure Container Instance Group, please refer to Microsoft document: [Container Groups - REST API (Azure Container Instances) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/container-instances/container-groups?view=rest-container-instances-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Container Groups - Get Outbound Network Dependencies Endpoints - REST API (Azure Container Instances) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/container-instances/container-groups/get-outbound-network-dependencies-endpoints?view=rest-container-instances-2025-09-01#code-try-0)
