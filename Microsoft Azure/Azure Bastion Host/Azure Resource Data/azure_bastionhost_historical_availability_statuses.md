# Introduction <a name="introduction"></a>
The API is used to list all historical availability transitions and impacting events for a single resource from Azure API Server (https://management.azure.com/). The response includes the details like id, name, type, location and properties.



The original Azure RESTful API used is <Availability Statuses - List>. 



For a complete list of available API resources for Azure Bastion Host, please refer to Microsoft document: [Availability Statuses - REST API (Azure Resource Health) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resourcehealth/availability-statuses?view=rest-resourcehealth-2024-02-01)

For the original Azure API detailed information, please refer to Microsoft document: [Availability Statuses - List - REST API (Azure Resource Health) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resourcehealth/availability-statuses/list?view=rest-resourcehealth-2024-02-01&tabs=HTTP)
