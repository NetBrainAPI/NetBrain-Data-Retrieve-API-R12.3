# Introduction <a name="introduction"></a>
The API is used to get the Azure SQL Managed Instance resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Managed Instances - Get>. 



For a complete list of available API resources for Azure SQL Managed Instance, please refer to Microsoft document: [Managed Instances - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/managed-instances?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Managed Instances - Get - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/managed-instances/get?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
