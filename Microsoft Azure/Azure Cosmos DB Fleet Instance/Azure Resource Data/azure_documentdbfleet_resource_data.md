# Introduction <a name="introduction"></a>
The API is used to get the Azure Cosmos DB Fleet Instance resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Fleet - Get>. 



For a complete list of available API resources for Azure Cosmos DB Fleet Instance, please refer to Microsoft document: [Fleet - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/fleet?view=rest-cosmos-db-resource-provider-2025-10-15)

For the original Azure API detailed information, please refer to Microsoft document: [Fleet - Get - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/fleet/get?view=rest-cosmos-db-resource-provider-2025-10-15&tabs=HTTP)
