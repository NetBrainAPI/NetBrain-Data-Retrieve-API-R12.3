# Introduction <a name="introduction"></a>
The API is used to get the backend health of the specified application gateway in a resource group from Azure API Server (https://management.azure.com/). The response includes the details like backendAddressPools.



The original Azure RESTful API used is <Application Gateways - Backend Health>. 



For a complete list of available API resources for Azure Application Gateway , please refer to Microsoft document: [Application Gateways - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/application-gateways?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Application Gateways - Backend Health - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/application-gateways/backend-health?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01&tabs=HTTP)
