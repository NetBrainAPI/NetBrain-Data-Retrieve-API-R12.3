# Introduction <a name="introduction"></a>
The API is used to get all the application gateways in the current subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location and properties.



The original Azure RESTful API used is <Application Gateways - List All >. 



For a complete list of available API resources for Azure Application Gateway , please refer to Microsoft document: [Application Gateways - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/application-gateways?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Application Gateways - List All - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/application-gateways/list-all?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01&tabs=HTTP)
