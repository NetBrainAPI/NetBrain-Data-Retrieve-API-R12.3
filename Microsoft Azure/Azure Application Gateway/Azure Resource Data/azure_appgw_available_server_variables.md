# Introduction <a name="introduction"></a>
The API is used to list all available server variables from Azure API Server (https://management.azure.com/). The response includes the details of all available server variables.



The original Azure RESTful API used is <Application Gateways - List Available Server Variables>. 



For a complete list of available API resources for Azure Application Gateway , please refer to Microsoft document: [Application Gateways - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/application-gateways?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Application Gateways - List Available Server Variables - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/application-gateways/list-available-server-variables?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01&tabs=HTTP)
