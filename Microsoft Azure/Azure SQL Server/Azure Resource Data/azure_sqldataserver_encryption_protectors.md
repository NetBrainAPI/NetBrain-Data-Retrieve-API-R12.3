# Introduction <a name="introduction"></a>
The API is used to get a list of server encryption protectors from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Encryption Protectors - List By Server>. 



For a complete list of available API resources for Azure SQL Servers, please refer to Microsoft document: [Encryption Protectors - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/encryption-protectors?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Encryption Protectors - List By Server - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/encryption-protectors/list-by-server?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
