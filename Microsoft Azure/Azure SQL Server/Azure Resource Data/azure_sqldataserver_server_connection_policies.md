# Introduction <a name="introduction"></a>
The API is used to list connection policy from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Server Connection Policies - List By Server>. 



For a complete list of available API resources for Azure SQL Servers, please refer to Microsoft document: [Server Connection Policies - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/server-connection-policies?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Server Connection Policies - List By Server - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/server-connection-policies/list-by-server?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
