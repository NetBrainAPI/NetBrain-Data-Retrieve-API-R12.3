# Introduction <a name="introduction"></a>
The API is used to get a list of inaccessible databases in a logical server from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Databases - List Inaccessible By Server>. 



For a complete list of available API resources for Azure SQL Databases, please refer to Microsoft document: [Databases - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/databases?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Databases - List Inaccessible By Server - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/databases/list-inaccessible-by-server?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
