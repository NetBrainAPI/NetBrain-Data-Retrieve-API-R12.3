# Introduction <a name="introduction"></a>
The API is used to get a list of firewall rules from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type and properties.



The original Azure RESTful API used is <Firewall Rules - List By Server>. 



For a complete list of available API resources for Azure SQL Servers, please refer to Microsoft document: [Firewall Rules - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/firewall-rules?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Firewall Rules - List By Server - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/firewall-rules/list-by-server?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
