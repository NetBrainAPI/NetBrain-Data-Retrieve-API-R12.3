# Introduction <a name="introduction"></a>
The API is used to retrieve information about the vpnclient ipsec policy for P2S client of virtual network gateway in the specified resource group through Network resource provider from Azure API Server (https://management.azure.com/). The response includes the details like ipsecEncryption, ipsecIntegrity, ikeEncryption, ect.



The original Azure RESTful API used is <Virtual Network Gateways - Get Vpnclient Ipsec Parameters>. 



For a complete list of available API resources for Azure Virtual Network Gateway, please refer to Microsoft document: [Virtual Network Gateways - REST API (Azure Network Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateways?view=rest-network-gateway-2025-03-01&viewFallbackFrom=rest-network-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateways/get-vpnclient-ipsec-parameters?view=rest-network-gateway-2024-05-01
