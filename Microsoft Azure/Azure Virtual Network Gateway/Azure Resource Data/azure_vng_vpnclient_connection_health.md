# Introduction <a name="introduction"></a>
The API is used to get VPN client connection health detail per P2S client connection of the virtual network gateway in the specified resource group from Azure API Server (https://management.azure.com/). The response includes the details like vpnConnectionId, vpnConnectionDuration, vpnConnectionTime, publicIpAddress, privateIpAddress, ect.



The original Azure RESTful API used is <Virtual Network Gateways - Get Vpnclient Connection Health>. 



For a complete list of available API resources for Azure Virtual Network Gateway, please refer to Microsoft document: [Virtual Network Gateways - REST API (Azure Network Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateways?view=rest-network-gateway-2025-03-01&viewFallbackFrom=rest-network-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Network Gateways - Get Vpnclient Connection Health - REST API (Azure Network Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateways/get-vpnclient-connection-health?view=rest-network-gateway-2025-03-01&viewFallbackFrom=rest-network-gateway-2024-05-01&tabs=HTTP)
