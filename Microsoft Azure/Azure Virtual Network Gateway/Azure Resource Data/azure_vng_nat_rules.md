# Introduction <a name="introduction"></a>
The API is used to retrieves all nat rules for a particular virtual network gateway from Azure API Server (https://management.azure.com/). The response includes the details like name, id and properties.



The original Azure RESTful API used is <Virtual Network Gateway Nat Rules - List By Virtual Network Gateway>. 



For a complete list of available API resources for Azure Virtual Network Gateway, please refer to Microsoft document: [Virtual Network Gateway Nat Rules - REST API (Azure Network Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateway-nat-rules?view=rest-network-gateway-2025-03-01&viewFallbackFrom=rest-network-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Network Gateway Nat Rules - List By Virtual Network Gateway - REST API (Azure Network Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateway-nat-rules/list-by-virtual-network-gateway?view=rest-network-gateway-2025-03-01&viewFallbackFrom=rest-network-gateway-2024-05-01&tabs=HTTP)
