# Introduction <a name="introduction"></a>
The API is used to list all of the dedicated hosts in the specified dedicated host group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Dedicated Hosts - List By Host Group>. 



For a complete list of available API resources for Azure Dedicated Host Groups, please refer to Microsoft document: [Dedicated Hosts - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/dedicated-hosts?view=rest-compute-2024-11-04)

For the original Azure API detailed information, please refer to Microsoft document: [Dedicated Hosts - List By Host Group - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/dedicated-hosts/list-by-host-group?view=rest-compute-2024-11-04&tabs=HTTP#code-try-0)
