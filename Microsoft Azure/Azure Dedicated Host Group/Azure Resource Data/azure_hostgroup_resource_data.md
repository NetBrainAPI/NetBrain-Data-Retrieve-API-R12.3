# Introduction <a name="introduction"></a>
The API is used to get the Azure Dedicated Host Groups resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Dedicated Host Groups - Get>. 



For a complete list of available API resources for Azure Dedicated Host Groups, please refer to Microsoft document: [Dedicated Host Groups - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/dedicated-host-groups?view=rest-compute-2024-11-04)

For the original Azure API detailed information, please refer to Microsoft document: [Dedicated Host Groups - Get - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/dedicated-host-groups/get?view=rest-compute-2024-11-04&tabs=HTTP)
