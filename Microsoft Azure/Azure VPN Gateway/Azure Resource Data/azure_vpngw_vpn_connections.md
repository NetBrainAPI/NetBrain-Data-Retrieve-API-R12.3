# Introduction <a name="introduction"></a>
The API is used to retrieve all vpn connections for a particular virtual wan vpn gateway from Azure API Server (https://management.azure.com/). The response includes the details like name, id and properties.



The original Azure RESTful API used is <Vpn Connections - List By Vpn Gateway>. 



For a complete list of available API resources for Azure VPN Gateway, please refer to Microsoft document: [Vpn Connections - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/vpn-connections?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Vpn Connections - List By Vpn Gateway - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/vpn-connections/list-by-vpn-gateway?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01&tabs=HTTP)
