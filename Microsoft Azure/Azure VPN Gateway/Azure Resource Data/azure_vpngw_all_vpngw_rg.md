# Introduction <a name="introduction"></a>
The API is used to lists all the VpnGateways in a resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Vpn Gateways - List By Resource Group>. 



For a complete list of available API resources for Azure VPN Gateway, please refer to Microsoft document: [Vpn Gateways - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/vpn-gateways?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Vpn Gateways - List By Resource Group - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/vpn-gateways/list-by-resource-group?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01&tabs=HTTP)
