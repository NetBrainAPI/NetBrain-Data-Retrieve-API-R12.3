# Introduction <a name="introduction"></a>
The API is used to get the Azure Front Doors resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Profiles - Get>. 



For a complete list of available API resources for Azure Front Doors, please refer to Microsoft document: [Profiles - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/profiles?view=rest-cdn-2025-04-15)

For the original Azure API detailed information, please refer to Microsoft document: [Profiles - Get - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/profiles/get?view=rest-cdn-2025-04-15&tabs=HTTP)
