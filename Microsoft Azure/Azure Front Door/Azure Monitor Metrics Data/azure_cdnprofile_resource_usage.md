# Introduction <a name="introduction"></a>
The API is used to check the quota and actual usage of endpoints under the given Azure Front Door Standard or Azure Front Door Premium or CDN profile from Azure API Server (https://management.azure.com/). The response includes the details like resourceType, unit, currentValue and limit.



The original Azure RESTful API used is <Profiles - List Resource Usage>. 



For a complete list of available API resources for Azure Front Doors, please refer to Microsoft document: [Profiles - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/profiles?view=rest-cdn-2025-04-15)

For the original Azure API detailed information, please refer to Microsoft document: [Profiles - List Resource Usage - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/profiles/list-resource-usage?view=rest-cdn-2025-04-15&tabs=HTTP)
