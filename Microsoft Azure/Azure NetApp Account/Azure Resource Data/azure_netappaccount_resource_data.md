# Introduction <a name="introduction"></a>
The API is used to get the Azure NetApp Account resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Accounts - Get>. 



For a complete list of available API resources for Azure NetApp Account, please refer to Microsoft document: [Accounts - REST API (Azure Azure NetApp Files) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/netapp/accounts?view=rest-netapp-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Accounts - Get - REST API (Azure Azure NetApp Files) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/netapp/accounts/get?view=rest-netapp-2025-09-01&tabs=HTTP)
