# Introduction <a name="introduction"></a>
The API is used to contain data from encryption.keyVaultProperties as well as information about which private endpoint is used by each encryption sibling set from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Accounts - Get Change Key Vault Information>. 



For a complete list of available API resources for Azure NetApp Account, please refer to Microsoft document: [Accounts - REST API (Azure Azure NetApp Files) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/netapp/accounts?view=rest-netapp-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Accounts - Get Change Key Vault Information - REST API (Azure Azure NetApp Files) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/netapp/accounts/get-change-key-vault-information?view=rest-netapp-2025-09-01&tabs=HTTP)
