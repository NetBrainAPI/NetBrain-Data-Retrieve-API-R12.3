# Introduction <a name="introduction"></a>
The API is used to list all the private endpoint connections associated with the Redis Enterprise cluster from Azure API Server (https://management.azure.com/). The response includes the details like name id, type and properties.



The original Azure RESTful API used is <Private Endpoint Connections - List>. 



For a complete list of available API resources for Azure Managed Redis Instance, please refer to Microsoft document: [Private Endpoint Connections - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/private-endpoint-connections?view=rest-redis-redisenterprisecache-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Private Endpoint Connections - List - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/private-endpoint-connections/list?view=rest-redis-redisenterprisecache-2025-07-01&tabs=HTTP)
