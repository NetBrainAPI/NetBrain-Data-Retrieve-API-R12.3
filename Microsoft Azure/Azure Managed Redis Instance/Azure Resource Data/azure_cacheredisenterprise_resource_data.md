# Introduction <a name="introduction"></a>
The API is used to get the Azure Managed Redis Instance resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Redis Enterprise - Get>. 



For a complete list of available API resources for Azure Managed Redis Instance, please refer to Microsoft document: [Redis Enterprise - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/redis-enterprise?view=rest-redis-redisenterprisecache-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Redis Enterprise - Get - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/redis-enterprise/get?view=rest-redis-redisenterprisecache-2025-07-01&tabs=HTTP)
