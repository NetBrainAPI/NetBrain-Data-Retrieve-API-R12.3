# Introduction <a name="introduction"></a>
The API is used to get the private link resources that need to be created for a Redis Enterprise cluster from Azure API Server (https://management.azure.com/). The response includes the details like name id, type and properties.



The original Azure RESTful API used is <Private Link Resources - List By Cluster>. 



For a complete list of available API resources for Azure Managed Redis Instance, please refer to Microsoft document: [Private Link Resources - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/private-link-resources?view=rest-redis-redisenterprisecache-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Private Link Resources - List By Cluster - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/private-link-resources/list-by-cluster?view=rest-redis-redisenterprisecache-2025-07-01&tabs=HTTP)
