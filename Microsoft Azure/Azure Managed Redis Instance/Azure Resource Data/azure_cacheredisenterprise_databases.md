# Introduction <a name="introduction"></a>
The API is used to get all databases in the specified Redis Enterprise cluster from Azure API Server (https://management.azure.com/). The response includes the details like id, name, type and properties.



The original Azure RESTful API used is <Databases - List By Cluster>. 



For a complete list of available API resources for Azure Managed Redis Instance, please refer to Microsoft document: [Redis Enterprise - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/redis-enterprise?view=rest-redis-redisenterprisecache-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Databases - List By Cluster - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/databases/list-by-cluster?view=rest-redis-redisenterprisecache-2025-07-01&tabs=HTTP)
