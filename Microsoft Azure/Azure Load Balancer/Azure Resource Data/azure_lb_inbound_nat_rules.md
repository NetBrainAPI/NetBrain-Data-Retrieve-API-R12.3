# Introduction <a name="introduction"></a>
The API is used to get all the inbound NAT rules in a load balancer from Azure API Server (https://management.azure.com/). The response includes the details like name, id, and properties.



The original Azure RESTful API used is <Inbound Nat Rules - List>. 



For a complete list of available API resources for Azure Application Gateway , please refer to Microsoft document: [Inbound Nat Rules - REST API (Azure Load balancer) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/load-balancer/inbound-nat-rules?view=rest-load-balancer-2025-03-01&viewFallbackFrom=rest-load-balancer-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Inbound Nat Rules - List - REST API (Azure Load balancer) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/load-balancer/inbound-nat-rules/list?view=rest-load-balancer-2025-03-01&viewFallbackFrom=rest-load-balancer-2024-05-01&tabs=HTTP)
