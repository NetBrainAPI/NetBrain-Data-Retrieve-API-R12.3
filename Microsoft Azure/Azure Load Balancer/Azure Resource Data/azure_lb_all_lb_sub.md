# Introduction <a name="introduction"></a>
The API is used to get all the load balancers in the current subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Load Balancers - List All>. 



For a complete list of available API resources for Azure Application Gateway , please refer to Microsoft document: [Load Balancers - REST API (Azure Load balancer) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/load-balancer/load-balancers?view=rest-load-balancer-2025-03-01&viewFallbackFrom=rest-load-balancer-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Load Balancers - List All - REST API (Azure Load balancer) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/load-balancer/load-balancers/list-all?view=rest-load-balancer-2025-03-01&viewFallbackFrom=rest-load-balancer-2024-05-01&tabs=HTTP)
