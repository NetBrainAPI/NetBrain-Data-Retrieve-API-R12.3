# Introduction <a name="introduction"></a>
The API is used to get all the load balancer frontend IP configurations from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type and properties.



The original Azure RESTful API used is <Load Balancer Frontend IP Configurations - List>. 



For a complete list of available API resources for Azure Application Gateway , please refer to Microsoft document: [Load Balancer Frontend IP Configurations - REST API (Azure Load balancer) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/load-balancer/load-balancer-frontend-ip-configurations?view=rest-load-balancer-2025-03-01&viewFallbackFrom=rest-load-balancer-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Load Balancer Frontend IP Configurations - List - REST API (Azure Load balancer) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/load-balancer/load-balancer-frontend-ip-configurations/list?view=rest-load-balancer-2025-03-01&viewFallbackFrom=rest-load-balancer-2024-05-01&tabs=HTTP)
