# Introduction <a name="introduction"></a>
The API is used to get the details of the user provided function apps registered with a static site from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Static Sites - Get User Provided Function Apps For Static Site>. 



For a complete list of available API resources for Azure Static Web Apps, please refer to Microsoft document: [Static Sites - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/static-sites?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Static Sites - Get User Provided Function Apps For Static Site - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/static-sites/get-user-provided-function-apps-for-static-site?view=rest-appservice-2024-11-01&tabs=HTTP#code-try-0)
