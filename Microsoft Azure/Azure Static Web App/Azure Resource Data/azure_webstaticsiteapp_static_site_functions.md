# Introduction <a name="introduction"></a>
The API is used to get the functions of a static site from Azure API Server (https://management.azure.com/). The response includes the details like id, properties, etc.



The original Azure RESTful API used is <Static Sites - List Static Site Functions>. 



For a complete list of available API resources for Azure Static Web Apps, please refer to Microsoft document: [Static Sites - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/static-sites?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Static Sites - List Static Site Functions - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/static-sites/list-static-site-functions?view=rest-appservice-2024-11-01&tabs=HTTP#code-try-0)
