# Introduction <a name="introduction"></a>
The API is used to list FirewallStatusResource resources by Firewalls resource from Azure API Server (https://management.azure.com/). The response includes the details like id, name, properties, systemData and type.



The original Azure RESTful API used is <Firewall Status - List By Firewalls>. 



For a complete list of available API resources for Azure Cloud NGFW by Palo Alto Networks, please refer to Microsoft document: [Firewall Status - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/firewall-status?view=rest-cloudngfw-2025-10-08)

For the original Azure API detailed information, please refer to Microsoft document: [Firewall Status - List By Firewalls - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/firewall-status/list-by-firewalls?view=rest-cloudngfw-2025-10-08&tabs=HTTP)
