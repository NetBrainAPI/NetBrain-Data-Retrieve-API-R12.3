# Introduction <a name="introduction"></a>
The API is used to get support info for firewall from Azure API Server (https://management.azure.com/). The response includes the details like accountId, accountRegistered, productSerial, etc.



The original Azure RESTful API used is <Firewalls - get Support Info>. 



For a complete list of available API resources for Azure Cloud NGFW by Palo Alto Networks, please refer to Microsoft document: [Firewalls - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/firewalls?view=rest-cloudngfw-2025-10-08)

For the original Azure API detailed information, please refer to Microsoft document: [Firewalls - get Support Info - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/firewalls/get-support-info?view=rest-cloudngfw-2025-10-08&tabs=HTTP)
