# Introduction <a name="introduction"></a>
The API is used to get a FirewallStatusResource for a single resource from Azure API Server (https://management.azure.com/). The response includes the details like id, name, properties, systemData and type.



The original Azure RESTful API used is <Firewall Status - Get>. 



For a complete list of available API resources for Azure Cloud NGFW by Palo Alto Networks, please refer to Microsoft document: [Cloud NGFW REST API reference | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/?view=rest-cloudngfw-2025-05-23)

For the original Azure API detailed information, please refer to Microsoft document: [Firewall Status - Get - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/firewall-status/get?view=rest-cloudngfw-2025-05-23&tabs=HTTP#code-try-0)
