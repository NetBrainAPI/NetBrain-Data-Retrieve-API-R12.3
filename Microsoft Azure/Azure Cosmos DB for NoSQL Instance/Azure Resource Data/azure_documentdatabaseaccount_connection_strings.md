# Introduction <a name="introduction"></a>
The API is used to list the connection strings for the specified Azure Cosmos DB database account from Azure API Server (https://management.azure.com/). The response includes the details like connectionString, description, keyKind and type.



The original Azure RESTful API used is <Database Accounts - List Connection Strings>. 



For a complete list of available API resources for Azure Cosmos DB for NoSQL Instance, please refer to Microsoft document: [Database Accounts - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts?view=rest-cosmos-db-resource-provider-2025-10-15)

For the original Azure API detailed information, please refer to Microsoft document: [Database Accounts - List Connection Strings - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts/list-connection-strings?view=rest-cosmos-db-resource-provider-2025-10-15&tabs=HTTP)
