# Introduction <a name="introduction"></a>
The API is used to retrieve the usages (most recent data) for the given database account from Azure API Server (https://management.azure.com/). The response includes the details like unit, quotaPeriod, unit, limit, currentValue and name.

The original Azure RESTful API used is <Database Accounts - List Usages>. 



For a complete list of available API resources for Azure Cosmos DB for NoSQL Instance, please refer to Microsoft document: [Database Accounts - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts?view=rest-cosmos-db-resource-provider-2025-10-15)

For the original Azure API detailed information, please refer to Microsoft document: [Database Accounts - List Usages - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts/list-usages?view=rest-cosmos-db-resource-provider-2025-10-15&tabs=HTTP)
