# Introduction <a name="introduction"></a>
The API is used to retrieve metric definitions for the given database account from Azure API Server (https://management.azure.com/). The response includes the details like metricAvailabilities, primaryAggregationType, unit, resourceUri and name.

The original Azure RESTful API used is <Database Accounts - List Metric Definitions>. 



For a complete list of available API resources for Azure Cosmos DB for NoSQL Instance, please refer to Microsoft document: [Database Accounts - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts?view=rest-cosmos-db-resource-provider-2025-10-15)

For the original Azure API detailed information, please refer to Microsoft document: [Database Accounts - List Metric Definitions - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts/list-metric-definitions?view=rest-cosmos-db-resource-provider-2025-10-15&tabs=HTTP)
