# Introduction <a name="introduction"></a>
The API is used to list all of the virtual machines in the current subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Virtual Machines - List All>. 



For a complete list of available API resources for Azure Virtual Machine, please refer to Microsoft document: [Virtual Machines - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines?view=rest-compute-2024-11-04)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Machines - List All - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines/list-all?view=rest-compute-2024-11-04&tabs=HTTP)
