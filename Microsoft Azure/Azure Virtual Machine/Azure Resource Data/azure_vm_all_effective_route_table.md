# Introduction <a name="introduction"></a>
The API is used to get the Azure Virtual Machine effective route table information for the specified URL from Azure API Server (https://management.azure.com/). The response includes the details like source, state, addressPrefix, nextHopType and nextHopIpAddress.



The original Azure RESTful API used is <Network Interfaces - Get Effective Route Table>. 



For a complete list of available API resources for Azure Network Interfaces, please refer to Microsoft document: [Network Interfaces - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/network-interfaces?view=rest-virtualnetwork-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Network Interfaces - Get Effective Route Table - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/network-interfaces/get-effective-route-table?view=rest-virtualnetwork-2025-05-01&tabs=HTTP#code-try-0)
