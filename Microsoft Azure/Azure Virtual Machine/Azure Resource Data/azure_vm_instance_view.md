# Introduction <a name="introduction"></a>
The API is used to retrieve information about the run-time state of a virtual machine from Azure API Server (https://management.azure.com/). The response includes the details like computerName, osName, osVersion, vmAgent, disks, hyperVGeneration, assignedHost and statuses.



The original Azure RESTful API used is <Virtual Machines - Instance View>. 



For a complete list of available API resources for Azure Virtual Machine, please refer to Microsoft document: [Virtual Machines - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines?view=rest-compute-2024-11-04)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Machines - Instance View - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines/instance-view?view=rest-compute-2024-11-04&tabs=HTTP)
