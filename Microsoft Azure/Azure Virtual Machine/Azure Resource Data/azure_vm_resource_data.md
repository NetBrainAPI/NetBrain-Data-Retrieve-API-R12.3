# Introduction <a name="introduction"></a>
The API is used to retrieve information about the model view or the instance view of a virtual machine from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location and properties.



The original Azure RESTful API used is <Virtual Machines - Get>. 



For a complete list of available API resources for Azure Virtual Machine, please refer to Microsoft document: [Virtual Machines - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines?view=rest-compute-2024-11-04)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Machines - Get - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/virtual-machines/get?view=rest-compute-2024-11-04&tabs=HTTP)
