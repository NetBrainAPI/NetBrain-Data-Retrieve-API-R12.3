# Introduction <a name="introduction"></a>
The API is used to get all the public IP addresses in a subscription from Azure API Server (https://management.azure.com/). The response includes the details like id, name, location, properties and sku.



The original Azure RESTful API used is <Public IP Addresses - List All>. 



For a complete list of available API resources for Azure Public IP Addresses, please refer to Microsoft document: [Public IP Addresses - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/public-ip-addresses?view=rest-virtualnetwork-2025-03-01)

For the original Azure API detailed information, please refer to Microsoft document: [Public IP Addresses - List All - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/public-ip-addresses/list-all?view=rest-virtualnetwork-2025-03-01&tabs=HTTP)
