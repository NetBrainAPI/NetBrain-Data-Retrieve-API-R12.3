# Introduction <a name="introduction"></a>
The API is used to get all virtual networks in the current subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Virtual Networks - List All>. 



For a complete list of available API resources for Azure Virtual Network, please refer to Microsoft document: [Virtual Networks - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/virtual-networks?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: https://learn.microsoft.com/en-us/rest/api/virtualnetwork/virtual-networks/list-all?view=rest-virtualnetwork-2024-05-01
