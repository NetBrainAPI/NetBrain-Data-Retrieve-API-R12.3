# Introduction <a name="introduction"></a>
The API is used to list DNS resolver resource IDs linked to a virtual network from Azure API Server (https://management.azure.com/). The response includes the detail like id.



The original Azure RESTful API used is <Dns Resolvers - List By Virtual Network>. 



For a complete list of available API resources for Azure DNS Resolver Instance, please refer to Microsoft document: [Dns Resolvers - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolvers?view=rest-dns-dnsresolver-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Dns Resolvers - List By Virtual Network - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolvers/list-by-virtual-network?view=rest-dns-dnsresolver-2025-05-01&tabs=HTTP#code-try-0)
