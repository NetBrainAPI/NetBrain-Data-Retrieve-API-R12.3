# Introduction <a name="introduction"></a>
The API is used to get the current network topology by resource group from Azure API Server (https://management.azure.com/). The response includes the details like id, createdDateTime, lastModified, resources, etc.



The original Azure RESTful API used is <Network Watchers - Get Topology>. 



For a complete list of available API resources for Azure Network Watchers, please refer to Microsoft document: [Network Watchers - REST API (Azure Network Watcher) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-watcher/network-watchers?view=rest-network-watcher-2025-03-01&viewFallbackFrom=rest-network-watcher-2024-07-01)
