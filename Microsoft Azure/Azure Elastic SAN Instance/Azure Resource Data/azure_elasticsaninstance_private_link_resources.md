# Introduction <a name="introduction"></a>
The API is used to get the private link resources that need to be created for a elastic San from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Private Link Resources - List By Elastic San>. 



For a complete list of available API resources for Azure Elastic SAN Instance, please refer to Microsoft document: [Private Link Resources - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/private-link-resources?view=rest-elasticsan-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Private Link Resources - List By Elastic San - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/private-link-resources/list-by-elastic-san?view=rest-elasticsan-2025-09-01&tabs=HTTP#code-try-0)
