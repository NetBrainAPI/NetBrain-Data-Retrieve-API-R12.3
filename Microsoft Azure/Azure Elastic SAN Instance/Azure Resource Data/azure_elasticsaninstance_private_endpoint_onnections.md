# Introduction <a name="introduction"></a>
The API is used to list all Private Endpoint Connections associated with the Elastic San from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Private Endpoint Connections - List>. 



For a complete list of available API resources for Azure Elastic SAN Instance, please refer to Microsoft document: [Private Endpoint Connections - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/private-endpoint-connections?view=rest-elasticsan-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Private Endpoint Connections - List - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/private-endpoint-connections/list?view=rest-elasticsan-2025-09-01&tabs=HTTP#code-try-0)
