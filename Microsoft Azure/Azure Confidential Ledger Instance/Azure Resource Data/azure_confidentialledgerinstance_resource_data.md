# Introduction <a name="introduction"></a>
The API is used to get the Azure Confidential Ledger Instance resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Ledger - Get>. 



For a complete list of available API resources for Azure Confidential Ledger Instance, please refer to Microsoft document: [Ledger - REST API (Azure Azure confidential ledger) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/confidentialledger/ledger?view=rest-resource-manager-confidentialledger-2022-05-13)

For the original Azure API detailed information, please refer to Microsoft document: [Ledger - Get - REST API (Azure Azure confidential ledger) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/confidentialledger/ledger/get?view=rest-resource-manager-confidentialledger-2022-05-13&tabs=HTTP#code-try-0)
