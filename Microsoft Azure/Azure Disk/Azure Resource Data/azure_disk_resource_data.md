# Introduction <a name="introduction"></a>
The API is used to get the Azure Disk resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Disks - Get>. 



For a complete list of available API resources for Azure Disk, please refer to Microsoft document: [Disks - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/disks?view=rest-compute-2025-04-01)

For the original Azure API detailed information, please refer to Microsoft document: [Disks - Get - REST API (Azure Compute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/compute/disks/get?view=rest-compute-2025-04-01&tabs=HTTP#code-try-0)
