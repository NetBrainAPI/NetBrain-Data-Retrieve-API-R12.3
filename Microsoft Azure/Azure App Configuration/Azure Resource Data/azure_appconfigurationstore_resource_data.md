# Introduction <a name="introduction"></a>
The API is used to get the Azure App Configuration resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Configuration Stores - Get>. 



For a complete list of available API resources for Azure App Configuration, please refer to Microsoft document: [Configuration Stores - REST API (Azure App Configuration) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appconfiguration/configuration-stores?view=rest-appconfiguration-2024-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Configuration Stores - Get - REST API (Azure App Configuration) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appconfiguration/configuration-stores/get?view=rest-appconfiguration-2024-06-01&tabs=HTTP#code-try-0)
