# Introduction <a name="introduction"></a>
The API is used to retrieve a Redis cache's access keys from Azure API Server (https://management.azure.com/). The response includes the details like primaryKey and secondaryKey.



The original Azure RESTful API used is <Redis - List Keys>. 



For a complete list of available API resources for Azure Cache for Redis, please refer to Microsoft document: [Redis - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redis?view=rest-redis-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Redis - List Keys - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redis/list-keys?view=rest-redis-2024-11-01&tabs=HTTP#code-try-0)
