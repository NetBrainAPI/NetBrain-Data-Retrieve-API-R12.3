# Introduction <a name="introduction"></a>
The API is used to list all the location in the subscription from Azure API Server (https://management.azure.com/). The response includes the details like id, name, type, metadata, availabilityZoneMappings, ect.



The original Azure RESTful API used is <Subscriptions - List Locations>. 



For a complete list of available API resources for Azure Region, please refer to Microsoft document: [Subscriptions - List Locations - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/subscriptions/list-locations?view=rest-resources-2022-12-01&tabs=HTTP)

For the original Azure API detailed information, please refer to Microsoft document: [Subscriptions - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/subscriptions?view=rest-resources-2022-12-01)
