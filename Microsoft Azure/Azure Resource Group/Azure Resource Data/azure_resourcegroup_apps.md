# Introduction <a name="introduction"></a>
The API is used to get all web, mobile, and API apps in the specified resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Web Apps - List By Resource Group>. 



For a complete list of available API resources for Azure Apps, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - List By Resource Group - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/list-by-resource-group?view=rest-appservice-2024-11-01&tabs=HTTP#code-try-0)
