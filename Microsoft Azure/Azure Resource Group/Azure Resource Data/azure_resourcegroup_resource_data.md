# Introduction <a name="introduction"></a>
The API is used to get the Azure Resource Group resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Resource Groups - Get>. 



For a complete list of available API resources for Azure Resource Group, please refer to Microsoft document: [Resource Groups - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/resource-groups?view=rest-resources-2021-04-01)

For the original Azure API detailed information, please refer to Microsoft document: [Resource Groups - Get - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/resource-groups/get?view=rest-resources-2021-04-01#code-try-0)
