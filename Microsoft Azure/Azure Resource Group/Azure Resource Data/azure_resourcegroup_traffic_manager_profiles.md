# Introduction <a name="introduction"></a>
The API is used to list all Traffic Manager profiles within a resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Profiles - List By Resource Group>. 



For a complete list of available API resources for Azure Traffic Manager Profiles, please refer to Microsoft document: [Profiles - REST API (Azure Traffic Manager) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/trafficmanager/profiles?view=rest-trafficmanager-2022-04-01)

For the original Azure API detailed information, please refer to Microsoft document: [Profiles - List By Resource Group - REST API (Azure Traffic Manager) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/trafficmanager/profiles/list-by-resource-group?view=rest-trafficmanager-2022-04-01&tabs=HTTP#code-try-0)
