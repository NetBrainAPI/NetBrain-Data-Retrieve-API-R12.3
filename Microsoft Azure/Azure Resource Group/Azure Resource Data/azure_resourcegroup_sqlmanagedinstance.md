# Introduction <a name="introduction"></a>
The API is used to get a list of managed instances in a resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Managed Instances - List By Resource Group>. 



For a complete list of available API resources for Azure SQL Managed Instance, please refer to Microsoft document: [Managed Instances - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/managed-instances?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Managed Instances - List By Resource Group - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/managed-instances/list-by-resource-group?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
