# Introduction <a name="introduction"></a>
The API is used to list all Bastion Hosts in a resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Bastion Hosts - List By Resource Group>. 



For a complete list of available API resources for Azure Bastion Hosts, please refer to Microsoft document: [Bastion Hosts - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/bastion-hosts?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Bastion Hosts - List By Resource Group - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/bastion-hosts/list-by-resource-group?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-07-01&tabs=HTTP#code-try-0)
