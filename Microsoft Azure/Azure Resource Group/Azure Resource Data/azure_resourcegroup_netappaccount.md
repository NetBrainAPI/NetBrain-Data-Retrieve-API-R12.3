# Introduction <a name="introduction"></a>
The API is used to list and describe all NetApp accounts in the resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Accounts - List>. 



For a complete list of available API resources for Azure NetApp Account, please refer to Microsoft document: [Accounts - REST API (Azure Azure NetApp Files) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/netapp/accounts?view=rest-netapp-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Accounts - List - REST API (Azure Azure NetApp Files) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/netapp/accounts/list?view=rest-netapp-2025-09-01&tabs=HTTP)
