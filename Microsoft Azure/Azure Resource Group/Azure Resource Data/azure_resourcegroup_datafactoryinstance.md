# Introduction <a name="introduction"></a>
The API is used to list factories in a given resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Factories - List By Resource Group>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Factories - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/factories?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Factories - List By Resource Group - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/factories/list-by-resource-group?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
