# Introduction <a name="introduction"></a>
The API is used to list all of the protection policies within a resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Web Application Firewall Policies - List>. 



For a complete list of available API resources for Azure Application Gateway WAF Policies, please refer to Microsoft document: [Web Application Firewall Policies - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/web-application-firewall-policies?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Application Firewall Policies - List - REST API (Azure Application Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/application-gateway/web-application-firewall-policies/list?view=rest-application-gateway-2025-03-01&viewFallbackFrom=rest-application-gateway-2024-05-01&tabs=HTTP)
