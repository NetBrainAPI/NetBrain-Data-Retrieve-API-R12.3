# Introduction <a name="introduction"></a>
The API is used to list all Redis caches in a resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Redis - List By Resource Group>. 



For a complete list of available API resources for Azure Cache for Redis, please refer to Microsoft document: [Redis - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redis?view=rest-redis-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Redis - List By Resource Group - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redis/list-by-resource-group?view=rest-redis-2024-11-01&tabs=HTTP#code-try-0)
