# Introduction <a name="introduction"></a>
The API is used to list all of the Azure Front Door Standard, Azure Front Door Premium, and CDN profiles within a resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Availability Statuses - Get By Resource>. 



For a complete list of available API resources for Azure Front Doors, please refer to Microsoft document: [Profiles - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/profiles?view=rest-cdn-2025-04-15)

For the original Azure API detailed information, please refer to Microsoft document: [Profiles - List By Resource Group - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/profiles/list-by-resource-group?view=rest-cdn-2025-04-15&tabs=HTTP)
