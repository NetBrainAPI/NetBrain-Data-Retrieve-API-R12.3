# Introduction <a name="introduction"></a>
The API is used to get all private end point connections for a specific private link service from Azure API Server (https://management.azure.com/). The response includes the details like name, id and properties.



The original Azure RESTful API used is <Private Link Services - List Private Endpoint Connections>. 



For a complete list of available API resources for Azure Private Link Service , please refer to Microsoft document: [Private Link Services - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/private-link-services?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Private Link Services - List Private Endpoint Connections - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/private-link-services/list-private-endpoint-connections?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-05-01&tabs=HTTP)
