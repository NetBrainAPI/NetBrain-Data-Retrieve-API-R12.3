# Introduction <a name="introduction"></a>
The API is used to list all record sets in a DNS zone from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Record Sets - List All By Dns Zone>. 



For a complete list of available API resources for Azure Public DNS Zones, please refer to Microsoft document: [Record Sets - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/record-sets?view=rest-dns-2018-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Record Sets - List All By Dns Zone - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/record-sets/list-all-by-dns-zone?view=rest-dns-2018-05-01&tabs=HTTP)
