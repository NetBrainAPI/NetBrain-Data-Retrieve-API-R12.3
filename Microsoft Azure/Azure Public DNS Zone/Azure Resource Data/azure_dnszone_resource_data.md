# Introduction <a name="introduction"></a>
The API is used to get the Azure Public DNS Zone resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Zones - Get>. 



For a complete list of available API resources for Azure Public DNS Zones, please refer to Microsoft document: [Zones - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/zones?view=rest-dns-2018-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Zones - Get - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/zones/get?view=rest-dns-2018-05-01&tabs=HTTP)
