# Introduction <a name="introduction"></a>
The API is used to get the Azure SQL Server on Azure Virtual Machines resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Sql Virtual Machines - Get>. 



For a complete list of available API resources for Azure SQL Server on Azure Virtual Machines, please refer to Microsoft document: [Sql Virtual Machines - REST API (Azure SQL VM) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sqlvm/sql-virtual-machines?view=rest-sqlvm-2023-10-01)

For the original Azure API detailed information, please refer to Microsoft document: [Sql Virtual Machines - Get - REST API (Azure SQL VM) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sqlvm/sql-virtual-machines/get?view=rest-sqlvm-2023-10-01&tabs=HTTP#code-try-0)
