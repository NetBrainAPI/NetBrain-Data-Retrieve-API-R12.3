# Introduction <a name="introduction"></a>
The API is used to list usage stats from Azure API Server (https://management.azure.com/). The response includes the details like currentValue, id, limit, name and unit.



The original Azure RESTful API used is <Virtual Networks - List Usage>. 



For a complete list of available API resources for Azure Virtual Network, please refer to Microsoft document: [Virtual Networks - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/virtual-networks?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Networks - List Usage - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/virtual-networks/list-usage?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2024-05-01&tabs=HTTP)
