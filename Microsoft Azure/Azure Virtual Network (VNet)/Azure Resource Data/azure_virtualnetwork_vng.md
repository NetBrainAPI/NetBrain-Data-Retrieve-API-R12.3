# Introduction <a name="introduction"></a>
The API is used to get all virtual network gateways by resource group information from Azure API Server (https://management.azure.com/). The response includes the details like id, name, type, location, properties etc.



The original Azure RESTful API used is <Virtual Network Gateways - List>. 



For a complete list of available API resources for Azure Virtual Network Gateway, please refer to Microsoft document: [Virtual Network Gateways - REST API (Azure Network Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateways?view=rest-network-gateway-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Network Gateways - List - REST API (Azure Network Gateway) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-gateway/virtual-network-gateways/list?view=rest-network-gateway-2025-05-01&tabs=HTTP#code-try-0)
