# Introduction <a name="introduction"></a>
The API is used to list all Azure Firewalls in a resource group information from Azure API Server (https://management.azure.com/). The response includes the details like id, name, type, location, properties etc.



The original Azure RESTful API used is <Azure Firewalls - List>. 



For a complete list of available API resources for Azure Firewalls, please refer to Microsoft document: [Azure Firewalls - REST API (Azure Firewall) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/firewall/azure-firewalls?view=rest-firewall-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Azure Firewalls - List - REST API (Azure Firewall) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/firewall/azure-firewalls/list?view=rest-firewall-2025-05-01&tabs=HTTP&tryIt=true&source=docs#code-try-0)
