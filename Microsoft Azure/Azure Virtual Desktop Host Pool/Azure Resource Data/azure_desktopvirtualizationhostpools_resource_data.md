# Introduction <a name="introduction"></a>
The API is used to get the Azure Virtual Desktop Host Pool resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Host Pools - Get>. 



For a complete list of available API resources for Azure Virtual Desktop Host Pool, please refer to Microsoft document: [Host Pools - REST API (Azure Desktop Virtualization) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/desktopvirtualization/host-pools?view=rest-desktopvirtualization-2024-04-03)

For the original Azure API detailed information, please refer to Microsoft document: [Host Pools - Get - REST API (Azure Desktop Virtualization) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/desktopvirtualization/host-pools/get?view=rest-desktopvirtualization-2024-04-03&tabs=HTTP)
