# Introduction <a name="introduction"></a>
The API is used to get the config reference app settings and status of an app from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Web Apps - Get Site Connection String Key Vault References>. 



For a complete list of available API resources for Azure Function App, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - Get Site Connection String Key Vault References - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/get-site-connection-string-key-vault-references?view=rest-appservice-2024-11-01#code-try-0)
