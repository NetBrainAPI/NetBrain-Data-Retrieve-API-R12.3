# Introduction <a name="introduction"></a>
The API is used to get an app's deployment slots from Azure API Server (https://management.azure.com/). The response includes the details like name, id, properties, etc.



The original Azure RESTful API used is <Web Apps - List Slots>. 



For a complete list of available API resources for Azure Function App, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - List Slots - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/list-slots?view=rest-appservice-2024-11-01&tabs=HTTP#code-try-0)
