# Introduction <a name="introduction"></a>
The API is used to get site's Authentication / Authorization settings for apps via the V2 format from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc. The original Azure RESTful API used is <Web Apps - Get Auth Settings V 2 Without Secrets>.
