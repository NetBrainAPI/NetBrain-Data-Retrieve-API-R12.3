# Introduction <a name="introduction"></a>
The API is used to get data around private site access enablement and authorized Virtual Networks that can access the site from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Web Apps - Get Private Access>. 



For a complete list of available API resources for Azure Function App, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - Get Private Access - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/get-private-access?view=rest-appservice-2024-11-01#code-try-0)
