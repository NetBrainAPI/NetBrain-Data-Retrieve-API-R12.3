# Introduction <a name="introduction"></a>
The API is used to get hybrid connections configured for an app (or deployment slot, if specified) from Azure API Server (https://management.azure.com/). The response includes the details like name, id, properties, etc.



The original Azure RESTful API used is <Web Apps - List Relay Service Connections>. 



For a complete list of available API resources for Azure Function App, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - List Relay Service Connections - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/list-relay-service-connections?view=rest-appservice-2024-11-01#code-try-0)
