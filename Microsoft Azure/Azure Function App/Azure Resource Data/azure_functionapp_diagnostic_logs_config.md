# Introduction <a name="introduction"></a>
The API is used to get the logging configuration of an app from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Get Diagnostic Logs Configuration>. 



For a complete list of available API resources for Azure Function App, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - Get Diagnostic Logs Configuration - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/get-diagnostic-logs-configuration?view=rest-appservice-2024-11-01#code-try-0)
