# Introduction <a name="introduction"></a>
The API is used to query trigger runs from Azure API Server (https://management.azure.com/). The response includes the details like triggerName, triggerRunId, triggerType, properties, etc.



The original Azure RESTful API used is <Trigger Runs - Query By Factory>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Trigger Runs - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/trigger-runs?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Trigger Runs - Query By Factory - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/trigger-runs/query-by-factory?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
