# Introduction <a name="introduction"></a>
The API is used to get Data Plane access from Azure API Server (https://management.azure.com/). The response includes the details like policy, dataPlaneUrl, accessToken, etc.



The original Azure RESTful API used is <Factories - Get Data Plane Access>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Factories - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/factories?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Factories - Get Data Plane Access - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/factories/get-data-plane-access?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
