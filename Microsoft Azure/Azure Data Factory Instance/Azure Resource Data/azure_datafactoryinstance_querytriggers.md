# Introduction <a name="introduction"></a>
The API is used to query triggers. from Azure API Server (https://management.azure.com/). The response includes the details like id, name, type, properties, etc.



The original Azure RESTful API used is <Triggers - Query By Factory>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Triggers - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/triggers?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Triggers - Query By Factory - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/triggers/query-by-factory?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
