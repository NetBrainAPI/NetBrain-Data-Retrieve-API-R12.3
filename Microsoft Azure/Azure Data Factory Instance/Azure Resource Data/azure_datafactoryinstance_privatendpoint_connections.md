# Introduction <a name="introduction"></a>
The API is used to list Private endpoint connections from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <private End Point Connections - List By Factory>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Private End Point Connections - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/private-end-point-connections?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Private End Point Connections - List By Factory - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/private-end-point-connections/list-by-factory?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
