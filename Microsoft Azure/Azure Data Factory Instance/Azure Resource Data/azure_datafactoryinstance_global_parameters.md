# Introduction <a name="introduction"></a>
The API is used to list global parameters from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Global Parameters - List By Factory>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Global Parameters - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/global-parameters?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Global Parameters - List By Factory - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/global-parameters/list-by-factory?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
