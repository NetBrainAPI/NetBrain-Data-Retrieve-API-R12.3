# Introduction <a name="introduction"></a>
The API is used to get the Azure Data Factory Instance resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Factories - Get>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Factories - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/factories?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Factories - Get - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/factories/get?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
