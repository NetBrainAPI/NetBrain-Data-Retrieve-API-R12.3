# Introduction <a name="introduction"></a>
The API is used to get a list of Geo backup policies for the given database resource from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type and properties.



The original Azure RESTful API used is <Geo Backup Policies - List>. 



For a complete list of available API resources for Azure SQL Databases, please refer to Microsoft document: [Geo Backup Policies - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/geo-backup-policies?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Geo Backup Policies - List - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/geo-backup-policies/list?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
