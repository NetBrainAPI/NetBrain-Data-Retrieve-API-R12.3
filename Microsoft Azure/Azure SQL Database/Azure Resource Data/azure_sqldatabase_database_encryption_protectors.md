# Introduction <a name="introduction"></a>
The API is used to revalidate an existing encryption protector for a particular database from Azure API Server (https://management.azure.com/). The response includes the details like validation status.



The original Azure RESTful API used is <Database Encryption Protectors - Revalidate>. 



For a complete list of available API resources for Azure SQL Databases, please refer to Microsoft document: [Database Encryption Protectors - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/database-encryption-protectors?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Database Encryption Protectors - Revalidate - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/database-encryption-protectors/revalidate?view=rest-sql-2023-08-01&tabs=HTTP)
