# Introduction <a name="introduction"></a>
The API is used to list database schemas from Azure API Server (https://management.azure.com/). The response includes the details like name, id and type.



The original Azure RESTful API used is <Database Schemas - List By Database>. 



For a complete list of available API resources for Azure SQL Databases, please refer to Microsoft document: [Database Schemas - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/database-schemas?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Database Schemas - List By Database - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/database-schemas/list-by-database?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
