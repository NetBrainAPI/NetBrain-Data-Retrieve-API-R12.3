# Introduction <a name="introduction"></a>
The API is used to get a list of operations performed on the database from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type and properties.



The original Azure RESTful API used is <Database Operations - List By Database>. 



For a complete list of available API resources for Azure SQL Databases, please refer to Microsoft document: [Database Operations - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/database-operations?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Database Operations - List By Database - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/database-operations/list-by-database?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
