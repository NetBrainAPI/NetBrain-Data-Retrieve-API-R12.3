# Introduction <a name="introduction"></a>
The API is used to get a list of all servers in the subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Servers - List>. 



For a complete list of available API resources for Azure SQL Servers, please refer to Microsoft document: [Servers - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/servers?view=rest-sql-2023-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Servers - List - REST API (Azure SQL Database) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sql/servers/list?view=rest-sql-2023-08-01&tabs=HTTP#code-try-0)
