# Introduction <a name="introduction"></a>
The API is used to check the quota and actual usage of the CDN profiles under the given subscription from Azure API Server (https://management.azure.com/). The response includes the details like resourceType, unit, currentValue and limit.



The original Azure RESTful API used is <Resource Usage - List>. 



For a complete list of available API resources for Azure Front Doors, please refer to Microsoft document: [Resource Usage - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/resource-usage?view=rest-cdn-2025-04-15)

For the original Azure API detailed information, please refer to Microsoft document: [Resource Usage - List - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/resource-usage/list?view=rest-cdn-2025-04-15&tabs=HTTP)
