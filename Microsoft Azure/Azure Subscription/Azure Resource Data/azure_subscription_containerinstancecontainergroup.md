# Introduction <a name="introduction"></a>
The API is used to get a list of container groups in the specified subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Container Groups - List>. 



For a complete list of available API resources for Azure Container Instance Group, please refer to Microsoft document: [Container Groups - REST API (Azure Container Instances) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/container-instances/container-groups?view=rest-container-instances-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Container Groups - List - REST API (Azure Container Instances) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/container-instances/container-groups/list?view=rest-container-instances-2025-09-01&tabs=HTTP#code-try-0)
