# Introduction <a name="introduction"></a>
The API is used to get all the resource groups for a subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Resource Groups - List>. 



For a complete list of available API resources for Azure Subscriptions, please refer to Microsoft document: [Resource Groups - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/resource-groups?view=rest-resources-2021-04-01)

For the original Azure API detailed information, please refer to Microsoft document: [Resource Groups - List - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/resource-groups/list?view=rest-resources-2021-04-01#code-try-0)
