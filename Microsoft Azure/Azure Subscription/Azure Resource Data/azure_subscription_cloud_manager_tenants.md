# Introduction <a name="introduction"></a>
The API is used to list Cloud Manager Tenants from Azure API Server (https://management.azure.com/). The response includes the detail like tenantName .



The original Azure RESTful API used is <Palo Alto Networks Cloudngfw Operations - list Cloud Manager Tenants>. 



For a complete list of available API resources for Azure Cloud NGFW by Palo Alto Networks, please refer to Microsoft document: [Palo Alto Networks Cloudngfw Operations - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/palo-alto-networks-cloudngfw-operations?view=rest-cloudngfw-2025-10-08)

For the original Azure API detailed information, please refer to Microsoft document: [Palo Alto Networks Cloudngfw Operations - list Cloud Manager Tenants - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/palo-alto-networks-cloudngfw-operations/list-cloud-manager-tenants?view=rest-cloudngfw-2025-10-08&tabs=HTTP)
