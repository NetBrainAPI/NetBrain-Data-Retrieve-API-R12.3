# Introduction <a name="introduction"></a>
The API is used to get all DDoS protection plans in a subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Ddos Protection Plans - List>. 



For a complete list of available API resources for Azure DDoS Protection Plan, please refer to Microsoft document: [Ddos Protection Plans - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/ddos-protection-plans?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2025-01-01)

For the original Azure API detailed information, please refer to Microsoft document: [Ddos Protection Plans - List - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/ddos-protection-plans/list?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2025-01-01&tabs=HTTP#code-try-0)
