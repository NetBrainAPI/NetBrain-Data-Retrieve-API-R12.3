# Introduction <a name="introduction"></a>
The API is used to retrieve information about all Confidential Ledger resources under the given subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Ledger - List By Subscription>. 



For a complete list of available API resources for Azure Confidential Ledger Instance, please refer to Microsoft document: [Ledger - REST API (Azure Azure confidential ledger) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/confidentialledger/ledger?view=rest-resource-manager-confidentialledger-2022-05-13)

For the original Azure API detailed information, please refer to Microsoft document: [Ledger - List By Subscription - REST API (Azure Azure confidential ledger) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/confidentialledger/ledger/list-by-subscription?view=rest-resource-manager-confidentialledger-2022-05-13&tabs=HTTP#code-try-0)
