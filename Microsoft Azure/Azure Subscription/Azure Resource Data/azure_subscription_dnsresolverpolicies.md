# Introduction <a name="introduction"></a>
The API is used to list DNS resolver policies in all resource groups of a subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Dns Resolver Policies - List>. 



For a complete list of available API resources for Azure DNS Resolver Policies, please refer to Microsoft document: [Dns Resolver Policies - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolver-policies?view=rest-dns-dnsresolver-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Dns Resolver Policies - List - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolver-policies/list?view=rest-dns-dnsresolver-2025-05-01&tabs=HTTP#code-try-0)
