# Introduction <a name="introduction"></a>
The API is used to list LocalRulestackResource resources by subscription ID from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Local Rulestacks - List By Subscription>. 



For a complete list of available API resources for Azure Cloud NGFW by Palo Alto Networks, please refer to Microsoft document: [Local Rulestacks - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/local-rulestacks?view=rest-cloudngfw-2025-10-08)

For the original Azure API detailed information, please refer to Microsoft document: [Local Rulestacks - List By Subscription - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/local-rulestacks/list-by-subscription?view=rest-cloudngfw-2025-10-08&tabs=HTTP)
