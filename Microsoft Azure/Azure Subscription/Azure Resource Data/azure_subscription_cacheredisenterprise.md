# Introduction <a name="introduction"></a>
The API is used to list all Redis Enterprise clusters in the specified subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Redis Enterprise - List>. 



For a complete list of available API resources for Azure Managed Redis Instance, please refer to Microsoft document: [Redis Enterprise - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/redis-enterprise?view=rest-redis-redisenterprisecache-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Redis Enterprise - List - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/redis-enterprise/list?view=rest-redis-redisenterprisecache-2025-07-01&tabs=HTTP)
