# Introduction <a name="introduction"></a>
The API is used to list all available managed rule sets from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Managed Rule Sets - List>. 



For a complete list of available API resources for Azure Azure Front Door WAF Policies, please refer to Microsoft document: [Managed Rule Sets - REST API (Azure Front Door Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/frontdoor/webapplicationfirewall/managed-rule-sets?view=rest-application-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Managed Rule Sets - List - REST API (Azure Front Door Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/frontdoor/webapplicationfirewall/managed-rule-sets/list?view=rest-application-gateway-2024-05-01&tabs=HTTP)
