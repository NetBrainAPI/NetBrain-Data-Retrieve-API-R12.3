# Introduction <a name="introduction"></a>
The API is used to get the Azure Subscription resource information from Azure API Server (https://management.azure.com/). The response includes the details like id, authorizationSource, subscriptionId, tenantId, displayName, state, subscriptionPolicies, etc.



The original Azure RESTful API used is <Subscriptions - Get>. 



For a complete list of available API resources for Azure Front Doors, please refer to Microsoft document: [Subscriptions - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/subscriptions?view=rest-resources-2022-12-01)

For the original Azure API detailed information, please refer to Microsoft document: [Subscriptions - Get - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/subscriptions/get?view=rest-resources-2022-12-01&tabs=HTTP#code-try-0)
