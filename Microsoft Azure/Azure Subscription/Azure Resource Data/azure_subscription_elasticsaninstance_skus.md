# Introduction <a name="introduction"></a>
The API is used to list all the available Skus in the region and information related to them from Azure API Server (https://management.azure.com/). The response includes the details like name, resourceType, locations, capabilities, etc.



The original Azure RESTful API used is <Skus - List>. 



For a complete list of available API resources for Azure Elastic SAN Instance, please refer to Microsoft document: [Skus - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/skus?view=rest-elasticsan-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Skus - List - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/skus/list?view=rest-elasticsan-2025-09-01&tabs=HTTP#code-try-0)
