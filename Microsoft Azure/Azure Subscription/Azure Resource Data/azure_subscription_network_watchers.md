# Introduction <a name="introduction"></a>
The API is used to get all network watchers by subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Network Watchers - List All>. 



For a complete list of available API resources for Azure Network Watchers, please refer to Microsoft document: [Network Watchers - REST API (Azure Network Watcher) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-watcher/network-watchers?view=rest-network-watcher-2025-03-01&viewFallbackFrom=rest-network-watcher-2024-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Network Watchers - List All - REST API (Azure Network Watcher) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/network-watcher/network-watchers/list-all?view=rest-network-watcher-2025-03-01&viewFallbackFrom=rest-network-watcher-2024-07-01&tabs=HTTP#code-try-0)
