# Introduction <a name="introduction"></a>
The API is used to list all the Azure Cosmos DB database accounts available under the subscription (clarify it includes Azure Cosmos DB for MongoDB RU Account and Azure Cosmos DB for NoSQL services) from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.

The original Azure RESTful API used is <Database Accounts - List>. 



For a complete list of available API resources for Azure Cosmos DB Database Account, please refer to Microsoft document: [Database Accounts - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts?view=rest-cosmos-db-resource-provider-2025-10-15)

For the original Azure API detailed information, please refer to Microsoft document: [Database Accounts - List - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/database-accounts/list?view=rest-cosmos-db-resource-provider-2025-10-15&tabs=HTTP)
