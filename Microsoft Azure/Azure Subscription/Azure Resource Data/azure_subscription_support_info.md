# Introduction <a name="introduction"></a>
The API is used to list Support Info from Azure API Server (https://management.azure.com/). The response includes the details like accountId, accountRegistrationStatus, productSerial, etc.



The original Azure RESTful API used is <Palo Alto Networks Cloudngfw Operations - list Support Info>. 



For a complete list of available API resources for Azure Cloud NGFW by Palo Alto Networks, please refer to Microsoft document: [Palo Alto Networks Cloudngfw Operations - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/palo-alto-networks-cloudngfw-operations?view=rest-cloudngfw-2025-10-08)

For the original Azure API detailed information, please refer to Microsoft document: [Palo Alto Networks Cloudngfw Operations - list Support Info - REST API (Azure Cloud NGFW) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cloudngfw/palo-alto-networks-cloudngfw-operations/list-support-info?view=rest-cloudngfw-2025-10-08&tabs=HTTP)
