# Introduction <a name="introduction"></a>
The API is used to list the current availability status for all the resources in the subscription. from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Availability Statuses - List By Subscription Id>. 



For a complete list of available API resources for Azure Subscription, please refer to Microsoft document: [Availability Statuses - REST API (Azure Resource Health) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resourcehealth/availability-statuses?view=rest-resourcehealth-2024-02-01)

For the original Azure API detailed information, please refer to Microsoft document: [Availability Statuses - List By Subscription Id - REST API (Azure Resource Health) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resourcehealth/availability-statuses/list-by-subscription-id?view=rest-resourcehealth-2024-02-01&tabs=HTTP#code-try-0)
