# Introduction <a name="introduction"></a>
The API is used to list DNS resolvers in all resource groups of a subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Dns Resolvers - List>. 



For a complete list of available API resources for DNS Resolver Instances, please refer to Microsoft document: [Dns Resolvers - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolvers?view=rest-dns-dnsresolver-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Dns Resolvers - List - REST API (Azure DNS) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/dns/dnsresolver/dns-resolvers/list?view=rest-dns-dnsresolver-2025-05-01&tabs=HTTP#code-try-0)
