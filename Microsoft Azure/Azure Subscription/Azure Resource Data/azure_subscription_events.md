# Introduction <a name="introduction"></a>
The API is used to list service health events in the subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Events - List By Subscription Id>. 



For a complete list of available API resources for Azure Subscription, please refer to Microsoft document: [Events - REST API (Azure Resource Health) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resourcehealth/events?view=rest-resourcehealth-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Events - List By Subscription Id - REST API (Azure Resource Health) | Microsoft](https://learn.microsoft.com/en-us/rest/api/resourcehealth/events/list-by-subscription-id?view=rest-resourcehealth-2025-05-01&tabs=HTTP#code-try-0)
