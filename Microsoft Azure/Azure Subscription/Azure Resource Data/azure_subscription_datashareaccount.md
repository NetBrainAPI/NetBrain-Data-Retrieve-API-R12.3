# Introduction <a name="introduction"></a>
The API is used to list Accounts in a subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Accounts - List By Subscription>. 



For a complete list of available API resources for Azure Data Share Account, please refer to Microsoft document: [Accounts - REST API (Azure Data Share) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datashare/accounts?view=rest-datashare-2021-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Accounts - List By Subscription - REST API (Azure Data Share) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datashare/accounts/list-by-subscription?view=rest-datashare-2021-08-01&tabs=HTTP#code-try-0)
