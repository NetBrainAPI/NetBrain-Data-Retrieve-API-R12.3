# Introduction <a name="introduction"></a>
The API is used to get a list of ElasticSans in a subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Elastic Sans - List By Subscription>. 



For a complete list of available API resources for Azure Elastic SAN Instance, please refer to Microsoft document: [Elastic Sans - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/elastic-sans?view=rest-elasticsan-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Elastic Sans - List By Subscription - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/elastic-sans/list-by-subscription?view=rest-elasticsan-2025-09-01&tabs=HTTP#code-try-0)
