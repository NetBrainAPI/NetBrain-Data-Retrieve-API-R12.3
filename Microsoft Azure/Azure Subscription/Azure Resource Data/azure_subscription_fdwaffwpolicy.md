# Introduction <a name="introduction"></a>
The API is used to list all of the protection policies within a subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Policies - List By Subscription>. 



For a complete list of available API resources for Azure Front Door WAF Policies, please refer to Microsoft document: [Policies - REST API (Azure Front Door Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/frontdoor/webapplicationfirewall/policies?view=rest-application-gateway-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Policies - List By Subscription - REST API (Azure Front Door Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/frontdoor/webapplicationfirewall/policies/list-by-subscription?view=rest-application-gateway-2024-05-01&tabs=HTTP)
