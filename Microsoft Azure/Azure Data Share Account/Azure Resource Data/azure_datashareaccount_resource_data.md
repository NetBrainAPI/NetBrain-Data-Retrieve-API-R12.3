# Introduction <a name="introduction"></a>
The API is used to get the Azure Data Share Account resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.

The original Azure RESTful API used is <Accounts - Get>. 



For a complete list of available API resources for Azure Data Share Account, please refer to Microsoft document: [Accounts - REST API (Azure Data Share) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datashare/accounts?view=rest-datashare-2021-08-01)

For the original Azure API detailed information, please refer to Microsoft document: [Accounts - Get - REST API (Azure Data Share) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datashare/accounts/get?view=rest-datashare-2021-08-01&tabs=HTTP&tryIt=true&source=docs#code-try-0)
