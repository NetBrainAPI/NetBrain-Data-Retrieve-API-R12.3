# Introduction <a name="introduction"></a>
The API is used to get the outbound routes configured for the Virtual Hub on a particular connection from Azure API Server (https://management.azure.com/). The response includes the details like prefix, asPath and bgpCommunities.



The original Azure RESTful API used is <Virtual Hubs - Get Outbound Routes>. 



For a complete list of available API resources for Azure Virtual Hub, please refer to Microsoft document: [Virtual Hubs - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hubs?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Hubs - Get Outbound Routes - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hubs/get-outbound-routes?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01&tabs=HTTP)
