# Introduction <a name="introduction"></a>
The API is used to retrieves the details of all VirtualHubBgpConnections from Azure API Server (https://management.azure.com/). The response includes the details like name, id and properties.



The original Azure RESTful API used is <Virtual Hub Bgp Connections - List>. 



For a complete list of available API resources for Azure Virtual Hub, please refer to Microsoft document: [Virtual Hub Bgp Connections - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hub-bgp-connections?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Hub Bgp Connections - List - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hub-bgp-connections/list?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01&tabs=HTTP)
