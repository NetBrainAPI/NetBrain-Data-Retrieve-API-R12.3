# Introduction <a name="introduction"></a>
The API is used to retrieves the details of all VirtualHubIpConfigurations from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type and properties.



The original Azure RESTful API used is <Virtual Hub Ip Configuration - List>. 



For a complete list of available API resources for Azure Virtual Hub, please refer to Microsoft document: [Virtual Hub Ip Configuration - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hub-ip-configuration?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Hub Ip Configuration - List - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hub-ip-configuration/list?view=rest-virtualwan-2025-03-01&viewFallbackFrom=rest-virtualwan-2024-05-01&tabs=HTTP)
