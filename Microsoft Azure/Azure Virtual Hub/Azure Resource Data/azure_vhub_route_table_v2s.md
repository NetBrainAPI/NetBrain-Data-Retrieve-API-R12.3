# Introduction <a name="introduction"></a>
The API is used to retrieve the details of all VirtualHubRouteTableV2s from Azure API Server (https://management.azure.com/). The response includes the details like name, id, and properties.



The original Azure RESTful API used is <Virtual Hub Route Table V2s - List>. 



For a complete list of available API resources for Azure Virtual Hub, please refer to Microsoft document: [Virtual Hub Route Table V2s - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hub-route-table-v-2-s?view=rest-virtualwan-2025-03-01)

For the original Azure API detailed information, please refer to Microsoft document: [Virtual Hub Route Table V2s - List - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/virtual-hub-route-table-v-2-s/list?view=rest-virtualwan-2025-03-01&tabs=HTTP)
