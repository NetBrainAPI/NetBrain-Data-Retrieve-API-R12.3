# Introduction <a name="introduction"></a>
The API is used to get auth token for a container app from Azure API Server (https://management.azure.com/). The response includes the details like id, name, type, location, properties, etc.



The original Azure RESTful API used is <Container Apps - Get Auth Token>. 



For a complete list of available API resources for Azure Container App Instance, please refer to Microsoft document: [Container Apps - REST API (Azure Azure Container Apps) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/containerapps/container-apps?view=rest-resource-manager-containerapps-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Container Apps - Get Auth Token - REST API (Azure Azure Container Apps) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/containerapps/container-apps/get-auth-token?view=rest-resource-manager-containerapps-2025-07-01&tabs=HTTP#code-try-0)
