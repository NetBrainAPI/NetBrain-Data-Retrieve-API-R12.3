# Introduction <a name="introduction"></a>
The API is used to get the CPU, memory, and disk usage statistics for each Cassandra node in a cluster from Azure API Server (https://management.azure.com/). The response includes the details like reaperStatus, connectionErrors, dataCenters, etc.



The original Azure RESTful API used is <Cassandra Clusters - Status>. 



For a complete list of available API resources for Azure Managed Instance for Apache Cassandra Cluster, please refer to Microsoft document: [Cassandra Clusters - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/cassandra-clusters?view=rest-cosmos-db-resource-provider-2025-10-15)

For the original Azure API detailed information, please refer to Microsoft document: [Cassandra Clusters - Status - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/cassandra-clusters/status?view=rest-cosmos-db-resource-provider-2025-10-15&tabs=HTTP#code-try-0)
