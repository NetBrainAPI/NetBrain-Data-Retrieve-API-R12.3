# Introduction <a name="introduction"></a>
The API is used to get the Azure Managed Instance for Apache Cassandra Cluster resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Cassandra Clusters - Get>. 



For a complete list of available API resources for Azure Managed Instance for Apache Cassandra Cluster, please refer to Microsoft document: [Cassandra Clusters - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/cassandra-clusters?view=rest-cosmos-db-resource-provider-2025-10-15)

For the original Azure API detailed information, please refer to Microsoft document: [Cassandra Clusters - Get - REST API (Azure Cosmos DB Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cosmos-db-resource-provider/cassandra-clusters/get?view=rest-cosmos-db-resource-provider-2025-10-15&tabs=HTTP)
