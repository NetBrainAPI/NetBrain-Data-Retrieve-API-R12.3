# Introduction <a name="introduction"></a>
The API is used to list all the storage accounts available under the current subscription from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Storage Accounts - List>. 



For a complete list of available API resources for Azure Storage Account, please refer to Microsoft document: [Storage Accounts - REST API (Azure Storage Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/storagerp/storage-accounts?view=rest-storagerp-2024-01-01)

For the original Azure API detailed information, please refer to Microsoft document: [Storage Accounts - List - REST API (Azure Storage Resource Provider) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/storagerp/storage-accounts/list?view=rest-storagerp-2024-01-01&tabs=HTTP)
