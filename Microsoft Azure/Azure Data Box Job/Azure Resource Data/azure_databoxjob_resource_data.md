# Introduction <a name="introduction"></a>
The API is used to get the Azure Data Box Job resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Jobs - Get>. 



For a complete list of available API resources for Azure Data Box Job, please refer to Microsoft document: [Jobs - REST API (Azure Data Box) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/databox/jobs?view=rest-databox-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Jobs - Get - REST API (Azure Data Box) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/databox/jobs/get?view=rest-databox-2025-07-01&tabs=HTTP)
