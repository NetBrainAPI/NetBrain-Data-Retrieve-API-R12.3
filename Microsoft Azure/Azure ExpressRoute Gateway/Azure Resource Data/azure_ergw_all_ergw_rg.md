# Introduction <a name="introduction"></a>
The API is used to list ExpressRoute gateways in the current resource group from Azure API Server (https://management.azure.com/). The response includes the details like name, id, location, type and properties.



The original Azure RESTful API used is <Express Route Gateways - List By Resource Group>. 



For a complete list of available API resources for Azure ExpressRoute Gateway, please refer to Microsoft document: [Express Route Gateways - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-gateways?view=rest-expressroute-2019-12-01)

For the original Azure API detailed information, please refer to Microsoft document: [Express Route Gateways - List By Resource Group - REST API (Azure ExpressRoute) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/expressroute/express-route-gateways/list-by-resource-group?view=rest-expressroute-2019-12-01&tabs=HTTP)
