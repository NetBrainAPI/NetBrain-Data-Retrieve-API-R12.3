# Introduction <a name="introduction"></a>
The API is used to list ExpressRouteConnections from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Express Route Connections - List>. 



For a complete list of available API resources for Azure ExpressRoute Gateway, please refer to Microsoft document: [Express Route Connections - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/express-route-connections?view=rest-virtualwan-2025-05-01)

For the original Azure API detailed information, please refer to Microsoft document: [Express Route Connections - List - REST API (Azure Virtual WAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualwan/express-route-connections/list?view=rest-virtualwan-2025-05-01&viewFallbackFrom=rest-virtualwan-2025-03-01&tabs=HTTP&tryIt=true&source=docs#code-try-0)
