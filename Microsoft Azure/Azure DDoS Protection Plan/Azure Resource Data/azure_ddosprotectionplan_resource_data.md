# Introduction <a name="introduction"></a>
The API is used to get the DDoS Protection Plan resource information from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Ddos Protection Plans - Get>. 



For a complete list of available API resources for Azure DDos Protection Plan, please refer to Microsoft document: [Ddos Protection Plans - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/ddos-protection-plans?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2025-01-01)

For the original Azure API detailed information, please refer to Microsoft document: [Ddos Protection Plans - Get - REST API (Azure Virtual Networks) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/virtualnetwork/ddos-protection-plans/get?view=rest-virtualnetwork-2025-03-01&viewFallbackFrom=rest-virtualnetwork-2025-01-01&tabs=HTTP#code-try-0)
