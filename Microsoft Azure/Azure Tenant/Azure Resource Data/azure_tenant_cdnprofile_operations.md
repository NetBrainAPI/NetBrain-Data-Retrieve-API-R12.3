# Introduction <a name="introduction"></a>
The API is used to list all of the available CDN REST API operations from Azure API Server (https://management.azure.com/). The response includes the details like name and display.



The original Azure RESTful API used is <Operations - List>. 



For a complete list of available API resources for Azure Front Doors, please refer to Microsoft document: [Operations - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/operations?view=rest-cdn-2025-04-15)

For the original Azure API detailed information, please refer to Microsoft document: [Operations - List - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/operations/list?view=rest-cdn-2025-04-15&tabs=HTTP)
