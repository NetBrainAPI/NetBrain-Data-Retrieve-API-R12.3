# Introduction <a name="introduction"></a>
The API is used to get the global Point of Presence (POP) locations used to deliver CDN content to end users from Azure API Server (https://management.azure.com/). The response includes the details like id, type, name, properties, etc.



The original Azure RESTful API used is <Edge Nodes - List>. 



For a complete list of available API resources for Azure Front Doors, please refer to Microsoft document: [Edge Nodes - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/edge-nodes?view=rest-cdn-2025-04-15)

For the original Azure API detailed information, please refer to Microsoft document: [Edge Nodes - List - REST API (Azure CDN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/cdn/edge-nodes/list?view=rest-cdn-2025-04-15&tabs=HTTP#code-try-0)
