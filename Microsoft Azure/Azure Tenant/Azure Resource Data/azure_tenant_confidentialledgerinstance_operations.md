# Introduction <a name="introduction"></a>
The API is used to retrieve a list of available API operations under this Resource Provider from Azure API Server (https://management.azure.com/). The response includes the details like name, isDataAction and display.



The original Azure RESTful API used is <Operations - List>. 



For a complete list of available API resources for Azure Confidential Ledger Instance, please refer to Microsoft document: [Operations - REST API (Azure Azure confidential ledger) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/confidentialledger/operations?view=rest-resource-manager-confidentialledger-2022-05-13)

For the original Azure API detailed information, please refer to Microsoft document: [Operations - List - REST API (Azure Azure confidential ledger) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resource-manager/confidentialledger/operations/list?view=rest-resource-manager-confidentialledger-2022-05-13&tabs=HTTP#code-try-0)
