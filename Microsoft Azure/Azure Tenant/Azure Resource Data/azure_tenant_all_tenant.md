# Introduction <a name="introduction"></a>
The API is used to get the tenants for your account from Azure API Server (https://management.azure.com/). The response includes the details like id, tenantId, countryCode, displayName, domains, etc.



The original Azure RESTful API used is <Tenants - List>. 



For a complete list of available API resources for Azure Tenant, please refer to Microsoft document: [Tenants - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/tenants?view=rest-resources-2022-12-01)

For the original Azure API detailed information, please refer to Microsoft document: [Tenants - List - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/tenants/list?view=rest-resources-2022-12-01&tabs=HTTP)
