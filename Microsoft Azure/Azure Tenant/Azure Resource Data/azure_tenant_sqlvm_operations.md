# Introduction <a name="introduction"></a>
The API is used to list all of the available SQL Virtual Machine Rest API operations from Azure API Server (https://management.azure.com/). The response includes the details like isDataAction, name, display, etc.



The original Azure RESTful API used is <Operations - List>. 



For a complete list of available API resources for Azure SQL Server on Azure Virtual Machines, please refer to Microsoft document: [Operations - REST API (Azure SQL VM) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sqlvm/operations?view=rest-sqlvm-2023-10-01)

For the original Azure API detailed information, please refer to Microsoft document: [Operations - List - REST API (Azure SQL VM) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/sqlvm/operations/list?view=rest-sqlvm-2023-10-01&tabs=HTTP#code-try-0)
