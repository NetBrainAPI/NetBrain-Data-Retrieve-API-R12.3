# Introduction <a name="introduction"></a>
The API is used to list all of the available REST API operations of the Microsoft.Cache provider from Azure API Server (https://management.azure.com/). The response includes the details like name, properties and display.



The original Azure RESTful API used is <Operations - List>. 



For a complete list of available API resources for Azure Managed Redis Instance, please refer to Microsoft document: [Operations - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/operations?view=rest-redis-redisenterprisecache-2025-07-01)

For the original Azure API detailed information, please refer to Microsoft document: [Operations - List - REST API (Azure Redis Cache) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/redis/redisenterprisecache/operations/list?view=rest-redis-redisenterprisecache-2025-07-01&tabs=HTTP)
