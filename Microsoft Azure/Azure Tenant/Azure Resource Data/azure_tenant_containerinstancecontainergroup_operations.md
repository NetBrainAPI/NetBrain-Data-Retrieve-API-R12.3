# Introduction <a name="introduction"></a>
The API is used to list the operations for Azure Container Instance service from Azure API Server (https://management.azure.com/). The response includes the details like name, origin and display.



The original Azure RESTful API used is <Operations - List>. 



For a complete list of available API resources for Azure Container Instance Group, please refer to Microsoft document: [Operations - REST API (Azure Container Instances) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/container-instances/operations?view=rest-container-instances-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Operations - List - REST API (Azure Container Instances) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/container-instances/operations/list?view=rest-container-instances-2025-09-01&tabs=HTTP#code-try-0)
