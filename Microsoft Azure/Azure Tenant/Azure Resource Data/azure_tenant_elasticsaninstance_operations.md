# Introduction <a name="introduction"></a>
The API is used to list the operations for the provider from Azure API Server (https://management.azure.com/). The response includes the details like name, isDataAction and display.



The original Azure RESTful API used is <Operations - List>. 



For a complete list of available API resources for Azure Elastic SAN Instance, please refer to Microsoft document: [Operations - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/operations?view=rest-elasticsan-2025-09-01)

For the original Azure API detailed information, please refer to Microsoft document: [Operations - List - REST API (Azure Elastic SAN) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/elasticsan/operations/list?view=rest-elasticsan-2025-09-01&tabs=HTTP#code-try-0)
