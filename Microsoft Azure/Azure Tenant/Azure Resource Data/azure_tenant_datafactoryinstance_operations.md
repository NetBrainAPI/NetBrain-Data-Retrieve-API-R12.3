# Introduction <a name="introduction"></a>
The API is used to list the available Azure Data Factory API operations from Azure API Server (https://management.azure.com/). The response includes the details like name, origin, display and properties.



The original Azure RESTful API used is <Operations - List>. 



For a complete list of available API resources for Azure Data Factory Instance, please refer to Microsoft document: [Operations - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/operations?view=rest-datafactory-2018-06-01)

For the original Azure API detailed information, please refer to Microsoft document: [Operations - List - REST API (Azure Data Factory) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/datafactory/operations/list?view=rest-datafactory-2018-06-01&tabs=HTTP#code-try-0)
