# Introduction <a name="introduction"></a>
The API is used to get all subscriptions for a tenant from Azure API Server (https://management.azure.com/). The response includes the details like id, tenantId, subscriptionId, displayName, etc.



The original Azure RESTful API used is <Subscriptions - List>. 



For a complete list of available API resources for Azure Tenant, please refer to Microsoft document: [Subscriptions - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/subscriptions?view=rest-resources-2022-12-01)

For the original Azure API detailed information, please refer to Microsoft document: [Subscriptions - List - REST API (Azure Resource Management) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/resources/subscriptions/list?view=rest-resources-2022-12-01&tabs=HTTP#code-try-0)
