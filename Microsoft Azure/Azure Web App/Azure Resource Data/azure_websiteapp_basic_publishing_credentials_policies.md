# Introduction <a name="introduction"></a>
The API is used to return whether Scm basic auth is allowed and whether Ftp is allowed for a given site from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, properties, etc.



The original Azure RESTful API used is <Web Apps - List Basic Publishing Credentials Policies>. 



For a complete list of available API resources for Azure Web App, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - List Basic Publishing Credentials Policies - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/list-basic-publishing-credentials-policies?view=rest-appservice-2024-11-01&tabs=HTTP#code-try-0)
