# Introduction <a name="introduction"></a>
The API is used to list continuous web jobs for an app, or a deployment slot from Azure API Server (https://management.azure.com/). The response includes the details like name, id, type, location, properties, etc.



The original Azure RESTful API used is <Web Apps - List Continuous Web Jobs>. 



For a complete list of available API resources for Azure Web App, please refer to Microsoft document: [Web Apps - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps?view=rest-appservice-2024-11-01)

For the original Azure API detailed information, please refer to Microsoft document: [Web Apps - List Continuous Web Jobs - REST API (Azure App Service) | Microsoft Learn](https://learn.microsoft.com/en-us/rest/api/appservice/web-apps/list-continuous-web-jobs?view=rest-appservice-2024-11-01#code-try-0)
